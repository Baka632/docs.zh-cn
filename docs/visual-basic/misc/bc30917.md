 <span data-ttu-id="93f09-104">当派生类没有声明构造函数时，Visual Basic 会尝试生成一个调用的隐式无参数构造函数 `MyBase.New()` 。</span><span class="sxs-lookup"><span data-stu-id="93f09-104">When a derived class does not declare a constructor, Visual Basic attempts to generate an implicit parameterless constructor that calls `MyBase.New()`.</span></span> <span data-ttu-id="93f09-105">如果基类中没有无需参数即可调用的可访问的构造函数，则 Visual Basic 无法生成隐式构造函数。</span><span class="sxs-lookup"><span data-stu-id="93f09-105">If there is no accessible constructor in the base class that can be called without arguments, Visual Basic cannot generate an implicit constructor.</span></span> <span data-ttu-id="93f09-106">在这种情况下，所需的构造函数标记有 <xref:System.ObsoleteAttribute> 特性，因此 Visual Basic 不能调用它。</span><span class="sxs-lookup"><span data-stu-id="93f09-106">In this case, the required constructor is marked with the <xref:System.ObsoleteAttribute> attribute, so Visual Basic cannot call it.</span></span>
