# <a name="is-operand-of-type-typename-can-only-be-compared-to-nothing-because-typename-is-a-nullable-type"></a><span data-ttu-id="c7607-102">类型“typename”的操作数“Is”只能与“Nothing”比较，因为“typename”是一个可以为 null 的类型</span><span class="sxs-lookup"><span data-stu-id="c7607-102">'Is' operand of type 'typename' can only be compared to 'Nothing', because 'typename' is a nullable type</span></span>

<span data-ttu-id="c7607-103">已将声明为可以为 null 的变量与表达式进行比较，而不是使用 `Nothing` 运算符与 `Is` 比较。</span><span class="sxs-lookup"><span data-stu-id="c7607-103">A variable declared as nullable has been compared to an expression other than `Nothing` by using the `Is` operator.</span></span>  
  
 <span data-ttu-id="c7607-104">**错误 ID：** BC32127</span><span class="sxs-lookup"><span data-stu-id="c7607-104">**Error ID:** BC32127</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c7607-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="c7607-105">To correct this error</span></span>
  
1. <span data-ttu-id="c7607-106">若要将可为 null 的值类型与的表达式进行比较，而不是 `Nothing` 使用 `Is` 运算符，请对 `GetType` 可以为 null 的值类型调用方法，并将结果与表达式进行比较，如下面的示例中所示。</span><span class="sxs-lookup"><span data-stu-id="c7607-106">To compare a nullable value type to an expression other than `Nothing` by using the `Is` operator, call the `GetType` method on the nullable value type and compare the result to the expression, as shown in the following example.</span></span>  
  
```vb  
Dim number? As Integer = 5  

If number IsNot Nothing Then  
  If number.GetType() Is Type.GetType("System.Int32") Then

  End If  
End If  
```
