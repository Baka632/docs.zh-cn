<span data-ttu-id="ce942-103">如果用于初始化数组的维数与声明中的维数不匹配，则编译器无法推断此数组的数据类型。</span><span class="sxs-lookup"><span data-stu-id="ce942-103">If the dimensions used to initialize an array do not match the dimensions in the declaration, the compiler cannot infer a data type for the array.</span></span> <span data-ttu-id="ce942-104">例如，下面的代码会导致此错误。</span><span class="sxs-lookup"><span data-stu-id="ce942-104">For example, the following code causes this error.</span></span>  
  
```vb  
' Valid. exampleArray1 is a one-dimensional array of integers.  
Dim exampleArray1() = New Integer() {1, 2, 3}  
' Not valid.  
'Dim exampleArray2(,) = New Integer() {1, 2, 3}  
'Dim exampleArray3(,) = New Integer() {}  
```
