# <a name="type-typename-cannot-inherit-from-a-type-nested-within-it"></a><span data-ttu-id="4c5d8-102">类型“\<typename>”不能从嵌套在它里面的类型继承</span><span class="sxs-lookup"><span data-stu-id="4c5d8-102">Type '\<typename>' cannot inherit from a type nested within it</span></span>

<span data-ttu-id="4c5d8-103">类或接口定义包括 [Inherits Statement](../language-reference/statements/inherits-statement.md) ，它指定了嵌套在它里面的一种类型。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-103">A class or interface definition includes an [Inherits Statement](../language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="4c5d8-104">继承必须采用线性方式，而不能采用循环方式。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="4c5d8-105">类型不能从它本身继承的类型继承。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="4c5d8-106">一个相关的限制为：类型不能从尚未定义的类型中继承。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="4c5d8-107">继承涉及到重用基类成员的能力，这反过来又要求这些成员已定义。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="4c5d8-108">因此，Visual Basic 无法编译如下面的示例所示的代码。</span><span class="sxs-lookup"><span data-stu-id="4c5d8-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>  
  
```vb  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
