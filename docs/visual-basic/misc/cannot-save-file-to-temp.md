# <a name="cannot-save-file-to-temp"></a><span data-ttu-id="02c02-102">无法将文件保存到 TEMP</span><span class="sxs-lookup"><span data-stu-id="02c02-102">Cannot save file to TEMP</span></span>

<span data-ttu-id="02c02-103">组件找不到名为 TEMP 目录，或者包含 TEMP 目录的驱动器或分区的空间不足，不能保存信息。</span><span class="sxs-lookup"><span data-stu-id="02c02-103">Either a component cannot find a directory named TEMP, or the drive or partition containing the TEMP directory lacks sufficient space to save information.</span></span>
