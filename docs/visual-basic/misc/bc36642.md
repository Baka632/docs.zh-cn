# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="56ad2-102">Option Strict On 要求使用“As”子句来声明其类型无法推断的每个 lambda 表达式参数</span><span class="sxs-lookup"><span data-stu-id="56ad2-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>

<span data-ttu-id="56ad2-103">已经声明了 lambda 表达式中的参数，但未使用 `As` 子句，并且将 `Option Strict` 设置为了“on”。</span><span class="sxs-lookup"><span data-stu-id="56ad2-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```vb  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="56ad2-104">如果可以推断出 `n` 的类型，则上一个声明有效。</span><span class="sxs-lookup"><span data-stu-id="56ad2-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="56ad2-105">例如，如果将上一个 lambda 表达式分配到函数委托 `Del`：</span><span class="sxs-lookup"><span data-stu-id="56ad2-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="56ad2-106">现在可以从参数 `n` 推断出 `p`的类型：</span><span class="sxs-lookup"><span data-stu-id="56ad2-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```vb  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="56ad2-107">**错误 ID：** BC36642</span><span class="sxs-lookup"><span data-stu-id="56ad2-107">**Error ID:** BC36642</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="56ad2-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="56ad2-108">To correct this error</span></span>  
  
- <span data-ttu-id="56ad2-109">将 `As` 子句添加到参数声明：</span><span class="sxs-lookup"><span data-stu-id="56ad2-109">Add an `As` clause to the parameter declaration:</span></span>  
  
    ```vb  
    Dim increment3 = Function (n As Integer) n + 1  
    ```
