## <a name="example"></a><span data-ttu-id="28c76-109">示例</span><span class="sxs-lookup"><span data-stu-id="28c76-109">Example</span></span>  

 <span data-ttu-id="28c76-110">下面的类包含一个需要整数参数的默认属性 `defaultProp`。</span><span class="sxs-lookup"><span data-stu-id="28c76-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="28c76-111">由于该默认属性需要参数，因此以下声明将导致错误。</span><span class="sxs-lookup"><span data-stu-id="28c76-111">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```vb

```
