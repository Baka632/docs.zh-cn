# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>如果文件不存在，则无法查询文件信息
试图查询 <xref:System.IO.FileInfo> 对象以了解某个不存在的文件的相关信息。 通常这是由于提供文件名时出现了差错。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查你所提供的文件名是否正确。
