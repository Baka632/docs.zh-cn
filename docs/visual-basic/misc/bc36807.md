<span data-ttu-id="a5a09-103">针对不属于类型 <xref:System.Xml.Linq.XElement>、 <xref:System.Xml.Linq.XDocument>或 `IEnumerable(Of XElement)`的对象引用了一个 XML 子元素。</span><span class="sxs-lookup"><span data-stu-id="a5a09-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="a5a09-104">有关详细信息，请参阅 [XML Child Axis Property](../language-reference/xml-axis/xml-child-axis-property.md)。</span><span class="sxs-lookup"><span data-stu-id="a5a09-104">For more information, see [XML Child Axis Property](../language-reference/xml-axis/xml-child-axis-property.md).</span></span>

  
 <span data-ttu-id="a5a09-105">**错误 ID:** BC36807</span><span class="sxs-lookup"><span data-stu-id="a5a09-105">**Error ID:** BC36807</span></span>
