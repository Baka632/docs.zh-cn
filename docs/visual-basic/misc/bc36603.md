# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="89a74-102">多维数组无法转换为表达式树</span><span class="sxs-lookup"><span data-stu-id="89a74-102">Multi-dimensional array cannot be converted to an expression tree</span></span>

<span data-ttu-id="89a74-103">大多数表达式可转换为表达式树，但多维数组无法转换。</span><span class="sxs-lookup"><span data-stu-id="89a74-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="89a74-104">例如，下列代码会导致此错误：</span><span class="sxs-lookup"><span data-stu-id="89a74-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
