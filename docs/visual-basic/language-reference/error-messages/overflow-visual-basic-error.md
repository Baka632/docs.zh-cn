# <a name="overflow-visual-basic-error"></a>溢出（Visual Basic 错误）

文本表示其所分配到的数据类型的限制之外的值。  
  
 **错误 ID：** BC30036  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请查看目标数据类型的值范围，然后重写文本以符合该范围。
