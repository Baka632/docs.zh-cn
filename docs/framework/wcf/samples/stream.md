## <a name="processing-data-as-it-is-streamed"></a><span data-ttu-id="6ad4f-138">处理经过流处理的数据</span><span class="sxs-lookup"><span data-stu-id="6ad4f-138">Processing Data As It Is Streamed</span></span>  

 <span data-ttu-id="6ad4f-139">`GetStream`、`UploadStream` 和 `EchoStream` 操作都可以处理直接从文件发送数据或将接收的数据直接保存到文件的情况。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-139">The operations `GetStream`, `UploadStream` and `EchoStream` all deal with sending data directly from a file or saving received data directly to a file.</span></span> <span data-ttu-id="6ad4f-140">但是，在某些情况下，需要发送或接收大量数据，并对正在发送或接收的数据块执行某种处理。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-140">However in some cases, there is a requirement to send or receive large amounts of data and perform some processing on chunks of the data as it is being sent or received.</span></span> <span data-ttu-id="6ad4f-141">解决类似方案的一种方法是编写一个自定义流（一个派生自 <xref:System.IO.Stream> 的类）来处理读取或写入的数据。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-141">One way to address such scenarios is to write a custom stream (a class that derives from <xref:System.IO.Stream>) that processes data as it is read or written.</span></span> <span data-ttu-id="6ad4f-142">`GetReversedStream` 操作和 `ReverseStream` 类就是类似方法的一个示例。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-142">The `GetReversedStream` operation and `ReverseStream` class are an example of this.</span></span>  
  
 <span data-ttu-id="6ad4f-143">`GetReversedStream` 创建并返回 `ReverseStream` 的新实例。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-143">`GetReversedStream` creates and returns a new instance of `ReverseStream`.</span></span> <span data-ttu-id="6ad4f-144">当系统从该 `ReverseStream` 对象中进行读取时，会发生实际的处理。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-144">The actual processing happens as the system reads from that `ReverseStream` object.</span></span> <span data-ttu-id="6ad4f-145">`ReverseStream.Read` 实现从基础文件中读取字节块区，反转字节，然后返回经过反转的字节。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-145">The `ReverseStream.Read` implementation reads a chunk of bytes from the underlying file, reverses them, then returns the reversed bytes.</span></span> <span data-ttu-id="6ad4f-146">这不会反转整个文件内容，而是一次仅反转一个字节块区。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-146">This does not reverse the entire file content; it reverses one chunk of bytes at a time.</span></span> <span data-ttu-id="6ad4f-147">下面的示例演示在流中读取或写入内容时如何进行流处理。</span><span class="sxs-lookup"><span data-stu-id="6ad4f-147">This is an example to show how you can perform stream processing as the content is being read or written from and to the stream.</span></span>  
  
```csharp
class ReverseStream : Stream  
{  
  
    FileStream inStream;  
    internal ReverseStream(string filePath)  
    {  
        //Opens the file and places a StreamReader around it.  
        inStream = File.OpenRead(filePath);  
    }  
  
    // Other methods removed for brevity.  
  
    public override int Read(byte[] buffer, int offset, int count)  
    {  
        int countRead=inStream.Read(buffer, offset,count);  
        ReverseBuffer(buffer, offset, countRead);  
        return countRead;  
    }  
  
    public override void Close()  
    {  
        inStream.Close();  
        base.Close();  
    }  
    protected override void Dispose(bool disposing)  
    {  
        inStream.Dispose();  
        base.Dispose(disposing);  
    }  
    void ReverseBuffer(byte[] buffer, int offset, int count)  
    {  
        int i, j;  
        for (i = offset, j = offset + count - 1; i < j; i++, j--)  
        {  
            byte currenti = buffer[i];  
            buffer[i] = buffer[j];  
            buffer[j] = currenti;  
        }  
  
    }  
}  
```
