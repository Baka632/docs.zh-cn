# <a name="calls-duration"></a>调用持续时间
计数器名称：调用持续时间  
  
## <a name="description"></a>说明  
 调用该操作的平均持续时间。 平均持续时间根据此公式计算：(N1-N0)/(D1-D0)。  
  
> [!WARNING]
> 在异步 WCF 服务上使用时，调用 Duration 计数器将始终返回-1。
