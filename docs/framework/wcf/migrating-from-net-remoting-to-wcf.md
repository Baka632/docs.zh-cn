### <a name="serialization-usage"></a><span data-ttu-id="1d7d9-179">序列化用法</span><span class="sxs-lookup"><span data-stu-id="1d7d9-179">Serialization Usage</span></span>  

 <span data-ttu-id="1d7d9-180">.NET 远程处理和 WCF 均使用序列化发送客户端和服务器之间的对象，但它们在这些重要的方面有所区别：</span><span class="sxs-lookup"><span data-stu-id="1d7d9-180">Both .NET Remoting and WCF use serialization to send objects between client and server, but they differ in these important ways:</span></span>
