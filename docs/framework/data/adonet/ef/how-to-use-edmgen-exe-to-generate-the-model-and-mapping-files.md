### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="32c62-116">使用 EdmGen.exe 为 Visual Basic 项目生成 School 模型</span><span class="sxs-lookup"><span data-stu-id="32c62-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1. <span data-ttu-id="32c62-117">创建 School 数据库。</span><span class="sxs-lookup"><span data-stu-id="32c62-117">Create the School database.</span></span> <span data-ttu-id="32c62-118">有关详细信息，请参阅 [创建 School 示例数据库](/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100))。</span><span class="sxs-lookup"><span data-stu-id="32c62-118">For more information, see [Creating the School Sample Database](/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100)).</span></span>  
  
2. <span data-ttu-id="32c62-119">在命令提示符下执行以下命令（无换行符）：</span><span class="sxs-lookup"><span data-stu-id="32c62-119">At the command prompt, execute the following command without line breaks:</span></span>

    ```console  
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
