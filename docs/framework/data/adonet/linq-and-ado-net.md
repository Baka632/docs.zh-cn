 <span data-ttu-id="10797-109">使用语言集成查询 (LINQ) ，开发人员可以在其应用程序代码中形成基于集的查询，而不必使用单独的查询语言。</span><span class="sxs-lookup"><span data-stu-id="10797-109">Language-Integrated Query (LINQ) enables developers
