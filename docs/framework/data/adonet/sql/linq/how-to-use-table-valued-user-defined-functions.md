## <a name="example"></a><span data-ttu-id="a913e-108">示例</span><span class="sxs-lookup"><span data-stu-id="a913e-108">Example</span></span>  

 <span data-ttu-id="a913e-109">下面的 SQL 函数显式声明其返回一个 `TABLE`。</span><span class="sxs-lookup"><span data-stu-id="a913e-109">The following SQL function explicitly states that it returns a `TABLE`.</span></span> <span data-ttu-id="a913e-110">因此，隐式定义了所返回的行集结构。</span><span class="sxs-lookup"><span data-stu-id="a913e-110">Therefore, the returned rowset structure is implicitly defined.</span></span>  
  
```sql
CREATE FUNCTION ProductsCostingMoreThan(@cost money)  
RETURNS TABLE  
AS  
RETURN  
    SELECT ProductID, UnitPrice  
    FROM Products  
    WHERE UnitPrice > @cost  
```
