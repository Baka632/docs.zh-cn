## <a name="symptom"></a><span data-ttu-id="65bca-106">症状</span><span class="sxs-lookup"><span data-stu-id="65bca-106">Symptom</span></span>  

 <span data-ttu-id="65bca-107">应用程序使用本地格式手动序列化 UTC <xref:System.DateTime> 实例：</span><span class="sxs-lookup"><span data-stu-id="65bca-107">An application is manually serializing a UTC <xref:System.DateTime> instance using a local format:</span></span>

### <a name="cause"></a><span data-ttu-id="65bca-108">原因</span><span class="sxs-lookup"><span data-stu-id="65bca-108">Cause</span></span>  

 <span data-ttu-id="65bca-109"><xref:System.DateTime.ToString%2A?displayProperty=nameWithType> 方法的“z”格式包括本地时区偏移量，例如，“+10:00”表示悉尼时间。</span><span class="sxs-lookup"><span data-stu-id="65bca-109">The 'z' format for the <xref:System.DateTime.ToString%2A?displayProperty=nameWithType> method includes the local time zone offset, for example, "+10:00" for Sydney time.</span></span> <span data-ttu-id="65bca-110">就这一点而言，只有 <xref:System.DateTime> 值是本地时间时，它才会得出有意义的结果。</span><span class="sxs-lookup"><span data-stu-id="65bca-110">As such, it will only produce a meaningful result if the value of the <xref:System.DateTime> is local.</span></span> <span data-ttu-id="65bca-111">如果该值是 UTC 时间，则 <xref:System.DateTime.ToString%2A?displayProperty=nameWithType> 包括本地时区偏移量，但是不显示或调整时区说明符。</span><span class="sxs-lookup"><span data-stu-id="65bca-111">If the value is UTC time, <xref:System.DateTime.ToString%2A?displayProperty=nameWithType> includes the local time zone offset, but it does not display or adjust the time zone specifier.</span></span>  
  
### <a name="resolution"></a><span data-ttu-id="65bca-112">解决方法</span><span class="sxs-lookup"><span data-stu-id="65bca-112">Resolution</span></span>

  
 <span data-ttu-id="65bca-115">还有一种利用正确序列化的 <xref:System.DateTime.Kind%2A> 属性序列化 <xref:System.DateTime> 的“o”格式，而不管实例是本地时间、UTC 时间还是未指定的时间：</span><span class="sxs-lookup"><span data-stu-id="65bca-115">There is also an "o" format that serializes a <xref:System.DateTime> making use of the <xref:System.DateTime.Kind%2A> property that serializes correctly regardless of whether the instance is local, UTC, or unspecified:</span></span>  
  
```csharp

Serialize(myDateTime.ToString("o"));  
```

## <a name="example"></a><span data-ttu-id="65bca-121">示例</span><span class="sxs-lookup"><span data-stu-id="65bca-121">Example</span></span>
