 <span data-ttu-id="f3d86-119">客户端可以通过以下两种方法来自定义视图：设置范围和筛选。</span><span class="sxs-lookup"><span data-stu-id="f3d86-119">The client has two ways of customizing the view: by scoping and by filtering.</span></span> <span data-ttu-id="f3d86-120">设置范围是指从基元素开始定义视图的范围，例如，应用程序可能希望仅查找桌面的直接子级，或者查找某个应用程序窗口的所有后代。</span><span class="sxs-lookup"><span data-stu-id="f3d86-120">Scoping is defining the extent of the view, starting from a base element: for example, the application might want to find only direct children of the desktop, or all descendants of an application window.</span></span> <span data-ttu-id="f3d86-121">筛选是指定义要包括在视图中的元素类型。</span><span class="sxs-lookup"><span data-stu-id="f3d86-121">Filtering is defining the types of elements that are to be included in the view.</span></span>
