## <a name="examples"></a><span data-ttu-id="fd2bd-307">示例</span><span class="sxs-lookup"><span data-stu-id="fd2bd-307">Examples</span></span>  
 <span data-ttu-id="fd2bd-308">以下命令将目录文件 MyCatalogFileName.cat 添加到系统组件和驱动程序数据库中。</span><span class="sxs-lookup"><span data-stu-id="fd2bd-308">The following command adds the catalog file MyCatalogFileName.cat to the system component and driver database.</span></span> <span data-ttu-id="fd2bd-309">如有必要阻止替换名为 `/u` 的现有目录文件，`MyCatalogFileName.cat` 选项会生成唯一名称。</span><span class="sxs-lookup"><span data-stu-id="fd2bd-309">The `/u` option generates a unique name if necessary to prevent replacing an existing catalog file named `MyCatalogFileName.cat`.</span></span>  
  
```console

```  
  
 <span data-ttu-id="fd2bd-310">以下命令通过使用最佳证书对文件进行自动签名。</span><span class="sxs-lookup"><span data-stu-id="fd2bd-310">The following command signs a file automatically by using the best certificate.</span></span>  
  

 <span data-ttu-id="fd2bd-314">以下命令通过使用位于 `My` 存储中的证书对文件进行签名，该证书的主题名为 `My Company Certificate`。</span><span class="sxs-lookup"><span data-stu-id="fd2bd-314">The following command signs a file by using a certificate located in the `My` store that has a subject name of `My Company Certificate`.</span></span>
