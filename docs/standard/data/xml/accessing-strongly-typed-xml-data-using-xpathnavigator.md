## <a name="obtain-typed-values-using-valueas-properties"></a><span data-ttu-id="b5779-165">使用 ValueAs 属性获取类型化值</span><span class="sxs-lookup"><span data-stu-id="b5779-165">Obtain Typed Values Using ValueAs Properties</span></span>  

 <span data-ttu-id="b5779-166">节点的类型化值可以通过访问 <xref:System.Xml.XPath.XPathNavigator.TypedValue%2A> 的 <xref:System.Xml.XPath.XPathNavigator> 属性进行检索。</span><span class="sxs-lookup"><span data-stu-id="b5779-166">The typed value of a node can be retrieved by accessing the <xref:System.Xml.XPath.XPathNavigator.TypedValue%2A> property of the <xref:System.Xml.XPath.XPathNavigator>.</span></span> <span data-ttu-id="b5779-167">在某些情况下，可能需要将节点的类型化值转换为其他类型。</span><span class="sxs-lookup"><span data-stu-id="b5779-167">In certain cases you may want to convert the typed value of a node to a different type.</span></span> <span data-ttu-id="b5779-168">常见的示例是从 XML 节点获取数值。</span><span class="sxs-lookup"><span data-stu-id="b5779-168">A common example is to get a numeric value from an XML node.</span></span> <span data-ttu-id="b5779-169">例如，考虑以下未经过验证和非类型化的 XML 文档。</span><span class="sxs-lookup"><span data-stu-id="b5779-169">For example, consider the following unvalidated and untyped XML document.</span></span>

 <span data-ttu-id="b5779-171">但是，仍可以使用 <xref:System.Xml.XPath.XPathItem.ValueAs%2A>、<xref:System.Xml.XPath.XPathNavigator.ValueAsDouble%2A>、<xref:System.Xml.XPath.XPathNavigator.ValueAsInt%2A> 或 <xref:System.Xml.XPath.XPathNavigator.ValueAsLong%2A> 方法和属性以数值形式提取该值。</span><span class="sxs-lookup"><span data-stu-id="b5779-171">However, it is still possible to extract the value as a numeric value using the <xref:System.Xml.XPath.XPathItem.ValueAs%2A>, <xref:System.Xml.XPath.XPathNavigator.ValueAsDouble%2A>, <xref:System.Xml.XPath.XPathNavigator.ValueAsInt%2A>, or <xref:System.Xml.XPath.XPathNavigator.ValueAsLong%2A> method and properties.</span></span> <span data-ttu-id="b5779-172">以下示例说明如何使用 <xref:System.Xml.XPath.XPathItem.ValueAs%2A> 方法执行此类强制转换。</span><span class="sxs-lookup"><span data-stu-id="b5779-172">The following example illustrates performing such a cast using the <xref:System.Xml.XPath.XPathItem.ValueAs%2A> method.</span></span>
