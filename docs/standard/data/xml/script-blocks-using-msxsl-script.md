## <a name="msxslscript-element-definition"></a><span data-ttu-id="a01fb-111">msxsl:script 元素定义</span><span class="sxs-lookup"><span data-stu-id="a01fb-111">msxsl:script Element Definition</span></span>  
 <span data-ttu-id="a01fb-112">`msxsl:script` 元素是 Microsoft 对 XSLT 1.0 建议的扩展，包括以下定义：</span><span class="sxs-lookup"><span data-stu-id="a01fb-112">The `msxsl:script` element is a Microsoft extension to the XSLT 1.0 recommendation and has the following definition:</span></span>

  
 <span data-ttu-id="a01fb-113">`msxsl` 前缀绑定到 `urn:schemas-microsoft-com:xslt` 命名空间 URI 上。</span><span class="sxs-lookup"><span data-stu-id="a01fb-113">The `msxsl` prefix is bound to the `urn:schemas-microsoft-com:xslt` namespace URI.</span></span> <span data-ttu-id="a01fb-114">样式表必须包括 `xmlns:msxsl=urn:schemas-microsoft-com:xslt` 命名空间声明。</span><span class="sxs-lookup"><span data-stu-id="a01fb-114">The style sheet must include the `xmlns:msxsl=urn:schemas-microsoft-com:xslt` namespace declaration.</span></span>  
  
 <span data-ttu-id="a01fb-115">`language` 属性是可选项。</span><span class="sxs-lookup"><span data-stu-id="a01fb-115">The `language` attribute is optional.</span></span> <span data-ttu-id="a01fb-116">属性值是嵌入式代码块的代码语言。</span><span class="sxs-lookup"><span data-stu-id="a01fb-116">Its value is the code language of the embedded code block.</span></span> <span data-ttu-id="a01fb-117">该语言使用 <xref:System.CodeDom.Compiler.CodeDomProvider.CreateProvider%2A?displayProperty=nameWithType> 方法映射到相应的 CodeDOM 编译器。</span><span class="sxs-lookup"><span data-stu-id="a01fb-117">The language is mapped to the appropriate CodeDOM compiler using the <xref:System.CodeDom.Compiler.CodeDomProvider.CreateProvider%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="a01fb-118"><xref:System.Xml.Xsl.XslCompiledTransform> 类可以支持任何 Microsoft .NET 语言，假定在计算机上已安装了相应的提供程序并且已在 machine.config 文件的 system.codedom 部分进行注册。</span><span class="sxs-lookup"><span data-stu-id="a01fb-118">The <xref:System.Xml.Xsl.XslCompiledTransform> class can support any Microsoft .NET language, assuming the appropriate provider is installed on the machine and is registered in the system.codedom section of the machine.config file.</span></span> <span data-ttu-id="a01fb-119">如果未指定 `language` 属性，则默认语言为 JScript。</span><span class="sxs-lookup"><span data-stu-id="a01fb-119">If a `language` attribute is not specified, the language defaults to JScript.</span></span> <span data-ttu-id="a01fb-120">语言名称不区分大小写，所以“JavaScript”和“javascript”等效。</span><span class="sxs-lookup"><span data-stu-id="a01fb-120">The language name is not case-sensitive so 'JavaScript' and 'javascript' are equivalent.</span></span>  
  
 <span data-ttu-id="a01fb-121">`implements-prefix` 属性是必选项。</span><span class="sxs-lookup"><span data-stu-id="a01fb-121">The `implements-prefix` attribute is mandatory.</span></span> <span data-ttu-id="a01fb-122">此属性用于声明命名空间并将其与脚本块关联。</span><span class="sxs-lookup"><span data-stu-id="a01fb-122">This attribute is used to declare a namespace and associate it with the script block.</span></span> <span data-ttu-id="a01fb-123">此属性的值是表示命名空间的前缀。</span><span class="sxs-lookup"><span data-stu-id="a01fb-123">The value of this attribute is the prefix that represents the namespace.</span></span> <span data-ttu-id="a01fb-124">此前缀可以在样式表中的某一位置定义。</span><span class="sxs-lookup"><span data-stu-id="a01fb-124">This prefix can be defined somewhere in a style sheet.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="a01fb-125">当使用 `msxsl:script` 元素时，强烈建议无论使用何种语言，都应将脚本放置在 CDATA 节内。</span><span class="sxs-lookup"><span data-stu-id="a01fb-125">When using the `msxsl:script` element, we strongly recommend that the script, regardless of language, be placed inside a CDATA section.</span></span> <span data-ttu-id="a01fb-126">因为脚本可以包含给定语言的运算符、标识符或分隔符，如果不包含在 CDATA 节中，可能会错误地作为 XML 解释。</span><span class="sxs-lookup"><span data-stu-id="a01fb-126">Because the script can contain operators, identifiers, or delimiters for a given language, if it is not contained within a CDATA section, it has the potential of being misinterpreted as XML.</span></span> <span data-ttu-id="a01fb-127">以下 XML 显示可以放入代码的 CDATA 节的模板。</span><span class="sxs-lookup"><span data-stu-id="a01fb-127">The following XML shows a template of the CDATA section where code can be placed.</span></span>
