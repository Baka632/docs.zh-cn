# <a name="accessing-attributes-in-the-dom"></a><span data-ttu-id="fa614-102">访问 DOM 中的属性</span><span class="sxs-lookup"><span data-stu-id="fa614-102">Accessing Attributes in the DOM</span></span>

<span data-ttu-id="fa614-103">属性是元素的属性，不是元素的子级。</span><span class="sxs-lookup"><span data-stu-id="fa614-103">Attributes are properties of the element, not children of the element.</span></span> <span data-ttu-id="fa614-104">这一区别很重要，因为用来浏览 XML 文档对象模型 (DOM) 的同级、父级和子节点的方法不同。</span><span class="sxs-lookup"><span data-stu-id="fa614-104">This distinction is important because of the methods used to navigate sibling, parent, and child nodes of the XML Document Object Model (DOM).</span></span> <span data-ttu-id="fa614-105">例如，PreviousSibling  和 NextSibling  方法不用于从元素转到属性，也不用于在属性之间导航。</span><span class="sxs-lookup"><span data-stu-id="fa614-105">For example, the **PreviousSibling** and **NextSibling** methods are not used to navigate from an element to an attribute or between attributes.</span></span> <span data-ttu-id="fa614-106">相反，属性是元素属性，归元素所有，包含 OwnerElement  属性，而不是 parentNode  属性，并且有不同的导航方法。</span><span class="sxs-lookup"><span data-stu-id="fa614-106">Instead, an attribute is a property of an element and is owned by an element, has an **OwnerElement** property and not a **parentNode** property, and has distinct methods of navigation.</span></span>

## <a name="retrieving-all-attributes-into-a-collection"></a><span data-ttu-id="fa614-112">将所有属性检索到一个集合中</span><span class="sxs-lookup"><span data-stu-id="fa614-112">Retrieving All Attributes into a Collection</span></span>

<span data-ttu-id="fa614-113">如果需要将元素节点的所有属性添加到集合中，请调用 XmlElement.Attributes  属性。</span><span class="sxs-lookup"><span data-stu-id="fa614-113">If you want all the attributes of an element node put into a collection, call the **XmlElement.Attributes** property.</span></span> <span data-ttu-id="fa614-114">这会获取包含元素的所有属性的 XmlAttributeCollection  。</span><span class="sxs-lookup"><span data-stu-id="fa614-114">This gets the **XmlAttributeCollection** that contains all the attributes of an element.</span></span> <span data-ttu-id="fa614-115">XmlAttributeCollection  类继承自 XmlNamedNode  映射。</span><span class="sxs-lookup"><span data-stu-id="fa614-115">The **XmlAttributeCollection** class inherits from the **XmlNamedNode** map.</span></span> <span data-ttu-id="fa614-116">因此，集合中的方法和属性除了包括 XmlAttributeCollection  类专用方法和属性（如 ItemOf  属性或 Append  方法）外，还包括命名节点映射中的方法和属性。</span><span class="sxs-lookup"><span data-stu-id="fa614-116">Therefore, the methods and properties available on the collection include those available on a named node map in addition to methods and properties specific to the **XmlAttributeCollection** class, such as the **ItemOf** property or the **Append** method.</span></span> <span data-ttu-id="fa614-117">属性集合中的每一项都表示 XmlAttribute  节点。</span><span class="sxs-lookup"><span data-stu-id="fa614-117">Each item in the attribute collection represents an **XmlAttribute** node.</span></span> <span data-ttu-id="fa614-118">若要确定元素的属性数，请获取 XmlAttributeCollection  ，并使用 Count  属性确定集合中的 XmlAttribute  节点数。</span><span class="sxs-lookup"><span data-stu-id="fa614-118">To find the number of attributes on an element, get the **XmlAttributeCollection**, and use the **Count** property to see how many **XmlAttribute** nodes are in the collection.</span></span>

<span data-ttu-id="fa614-119">下面的代码示例展示了如何检索属性集合，以及如何通过对循环索引使用 Count  方法来循环访问集合。</span><span class="sxs-lookup"><span data-stu-id="fa614-119">The following code example shows how to retrieve an attribute collection and, using the **Count** method for the looping index, iterate over it.</span></span> <span data-ttu-id="fa614-120">然后，此代码显示如何从集合中检索单个属性并显示其值。</span><span class="sxs-lookup"><span data-stu-id="fa614-120">The code then shows how to retrieve a single attribute from the collection and display its value.</span></span>

```vb
Imports System.IO
Imports System.Xml

Public Class Sample

    Public Shared Sub Main()

        Dim doc As XmlDocument = New XmlDocument()
        doc.LoadXml("<book genre='novel' ISBN='1-861001-57-5' misc='sale item'>" & _
               "<title>The Handmaid's Tale</title>" & _
               "<price>14.95</price>" & _
               "</book>")

        ' Move to an element.
        Dim myElement As XmlElement = doc.DocumentElement

        ' Create an attribute collection from the element.
        Dim attrColl As XmlAttributeCollection = myElement.Attributes

        ' Show the collection by iterating over it.
        Console.WriteLine("Display all the attributes in the collection...")
        Dim i As Integer
        For i = 0 To attrColl.Count - 1
            Console.Write("{0} = ", attrColl.ItemOf(i).Name)
            Console.Write("{0}", attrColl.ItemOf(i).Value)
            Console.WriteLine()
        Next

        ' Retrieve a single attribute from the collection; specifically, the
        ' attribute with the name "misc".
        Dim attr As XmlAttribute = attrColl("misc")

        ' Retrieve the value from that attribute.
        Dim miscValue As String = attr.InnerXml

        Console.WriteLine("Display the attribute information.")
        Console.WriteLine(miscValue)

    End Sub
End Class
```

```csharp
using System;
using System.IO;
using System.Xml;

public class Sample
{

    public static void Main()
    {
        XmlDocument doc = new XmlDocument();
        doc.LoadXml("<book genre='novel' ISBN='1-861001-57-5' misc='sale item'>" +
                      "<title>The Handmaid's Tale</title>" +
                      "<price>14.95</price>" +
                      "</book>");

        // Move to an element.
        XmlElement myElement = doc.DocumentElement;

        // Create an attribute collection from the element.
        XmlAttributeCollection attrColl = myElement.Attributes;

        // Show the collection by iterating over it.
        Console.WriteLine("Display all the attributes in the collection...");
        for (int i = 0; i < attrColl.Count; i++)
        {
            Console.Write("{0} = ", attrColl[i].Name);
            Console.Write("{0}", attrColl[i].Value);
            Console.WriteLine();
        }

        // Retrieve a single attribute from the collection; specifically, the
        // attribute with the name "misc".
        XmlAttribute attr = attrColl["misc"];

        // Retrieve the value from that attribute.
        String miscValue = attr.InnerXml;

        Console.WriteLine("Display the attribute information.");
        Console.WriteLine(miscValue);

    }
}
```
