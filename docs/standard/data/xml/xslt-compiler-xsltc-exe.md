## <a name="examples"></a><span data-ttu-id="2f576-159">示例</span><span class="sxs-lookup"><span data-stu-id="2f576-159">Examples</span></span>  
 <span data-ttu-id="2f576-160">下面的命令编译样式表并创建一个名为 booksort.dll 的程序集。</span><span class="sxs-lookup"><span data-stu-id="2f576-160">The following command compiles the style sheet and creates an assembly named booksort.dll.</span></span>  
  
```console

```  
  
 <span data-ttu-id="2f576-161">下面的命令编译样式表并创建名称分别为 booksort.dll 和 booksort.pdb 的程序集和 PDB 文件。</span><span class="sxs-lookup"><span data-stu-id="2f576-161">The following command compiles the style sheet and creates an assembly and PDB file that are named booksort.dll and booksort.pdb respectively.</span></span>



