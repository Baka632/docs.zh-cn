# <a name="best-practices-for-regular-expressions-in-net"></a><span data-ttu-id="c46c4-103">.NET 中的正则表达式最佳做法</span><span class="sxs-lookup"><span data-stu-id="c46c4-103">Best practices for regular expressions in .NET</span></span>

<span data-ttu-id="c46c4-104">.NET 中的正则表达式引擎是一种功能强大而齐全的工具，它基于模式匹配（而不是比较和匹配文本）来处理文本。</span><span class="sxs-lookup"><span data-stu-id="c46c4-104">The regular expression engine in .NET is a powerful, full-featured tool that processes text based on pattern matches rather than on comparing and matching literal text.</span></span> <span data-ttu-id="c46c4-105">在大多数情况下，它可以快速、高效地执行模式匹配。</span><span class="sxs-lookup"><span data-stu-id="c46c4-105">In most cases, it performs pattern matching rapidly and efficiently.</span></span> <span data-ttu-id="c46c4-106">但在某些情况下，正则表达式引擎的速度似乎很慢。</span><span class="sxs-lookup"><span data-stu-id="c46c4-106">However, in some cases, the regular expression engine can appear to be very slow.</span></span> <span data-ttu-id="c46c4-107">在极端情况下，它甚至看似停止响应，因为它会用若干个小时甚至若干天处理相对小的输入。</span><span class="sxs-lookup"><span data-stu-id="c46c4-107">In extreme cases, it can even appear to stop responding as it processes a relatively small input over the course of hours or even days.</span></span>

<span data-ttu-id="c46c4-108">本主题概述开发人员为了确保其正则表达式实现最佳性能可以采纳的一些最佳做法。</span><span class="sxs-lookup"><span data-stu-id="c46c4-108">This topic outlines some of the best practices that developers can adopt to ensure that their regular expressions achieve optimal performance.</span></span>
