## <a name="symmetric-keys"></a><span data-ttu-id="74a3a-111">对称密钥</span><span class="sxs-lookup"><span data-stu-id="74a3a-111">Symmetric Keys</span></span>

 <span data-ttu-id="74a3a-116">为了将对称密钥和 IV 传送给远程方，通常使用不对称加密来加密对称密钥。</span><span class="sxs-lookup"><span data-stu-id="74a3a-116">To communicate a symmetric key and IV to a remote party, you would usually encrypt the symmetric key by using asymmetric encryption.</span></span> <span data-ttu-id="74a3a-117">通过不安全的网络发送密钥而不对其进行加密会很不安全，这是因为截获密钥和 IV 的任何人都能够解密您的数据。</span><span class="sxs-lookup"><span data-stu-id="74a3a-117">Sending the key across an insecure network without encrypting it is unsafe, because anyone who intercepts the key and IV can then decrypt your data.</span></span>


## <a name="asymmetric-keys"></a><span data-ttu-id="74a3a-125">非对称密钥</span><span class="sxs-lookup"><span data-stu-id="74a3a-125">Asymmetric Keys</span></span>

 <span data-ttu-id="74a3a-139">下面的代码示例创建 **RSA** 类的新实例，创建公钥/私钥对，并将公钥信息保存到 **RSAParameters** 结构。</span><span class="sxs-lookup"><span data-stu-id="74a3a-139">The following code example creates a new instance of the **RSA** class, creating a public/private key pair, and saves the public key information to an **RSAParameters** structure.</span></span>
