# <a name="breaking-changes-in-windows-presentation-framework-wpf"></a>Windows Presentation Framework (WPF) 中的中断性变更

WPF 支持已添加到版本 3.0 中的 .NET Core。 本文按引入了中断性变更的 .NET 版本列出了 WPF 的中断性变更。 如果要从 .NET Framework 或 .NET Core 的早期版本（3.0 或更高版本）升级 WPF 应用，本文能为你提供帮助。

本页记录了以下中断性变更：

| 重大更改 | 引入的版本 |
| - | :-: |
| [对于 WPF 和 WinForms 应用，OutputType 设置为 WinExe](#outputtype-set-to-winexe-for-wpf-and-winforms-apps) | 5.0 |
| [WinForms 和 WPF 应用使用 Microsoft.NET.Sdk](#winforms-and-wpf-apps-use-microsoftnetsdk) | 5.0 |
