        }
    }
}

## <a name="example-3"></a><span data-ttu-id="b9fff-159">示例 3</span><span class="sxs-lookup"><span data-stu-id="b9fff-159">Example 3</span></span>

<span data-ttu-id="b9fff-160">在下面的修订 `Person` 示例中， `other` 首先在方法中检查传递给复制构造函数的对象是否为 null `PassThroughNonNull` 。</span><span class="sxs-lookup"><span data-stu-id="b9fff-160">In the following revised `Person` example, the `other` object that is passed to the copy constructor is first checked for null in the `PassThroughNonNull` method.</span></span>

```csharp
public class Person
{
    public string Name { get; private set; }
    public int Age { get; private set; }

    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // Copy constructor
    public Person(Person other)
        : this(PassThroughNonNull(other).Name, other.Age)
    {
    }

    // Null check method
    private static Person PassThroughNonNull(Person person)
    {
        if (person == null)
            throw new ArgumentNullException(nameof(person));
        return person;
    }
}
```
