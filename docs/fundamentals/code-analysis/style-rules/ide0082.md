## <a name="example"></a><span data-ttu-id="3a402-120">示例</span><span class="sxs-lookup"><span data-stu-id="3a402-120">Example</span></span>

```csharp
// Code with violations

var n2 = typeof(int).Name;

// Fixed code
var n1 = nameof(T);
var n2 = nameof(Int32);
```

```vb
' Code with violations

```
