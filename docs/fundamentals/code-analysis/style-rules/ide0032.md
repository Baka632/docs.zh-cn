### <a name="example"></a><span data-ttu-id="ed04d-129">示例</span><span class="sxs-lookup"><span data-stu-id="ed04d-129">Example</span></span>

```csharp
// dotnet_style_prefer_auto_properties = true
private int Age { get; }

// dotnet_style_prefer_auto_properties = false
private int age;

public int Age
{
    get
    {
        return age;
    }
}
```

    End Get
End Property
