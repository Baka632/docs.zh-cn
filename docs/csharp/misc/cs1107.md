# <a name="compiler-error-cs1107"></a><span data-ttu-id="ff8dc-103">编译器错误 CS1107</span><span class="sxs-lookup"><span data-stu-id="ff8dc-103">Compiler Error CS1107</span></span>
<span data-ttu-id="ff8dc-104">参数只能具有一个“modifier name”修饰符。</span><span class="sxs-lookup"><span data-stu-id="ff8dc-104">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="ff8dc-105">对于参数修饰符（例如、、和） `this` `ref` `in` `out` 在参数定义中出现多次时，这是错误的。</span><span class="sxs-lookup"><span data-stu-id="ff8dc-105">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ff8dc-106">示例</span><span class="sxs-lookup"><span data-stu-id="ff8dc-106">Example</span></span>  
 <span data-ttu-id="ff8dc-107">下面的示例生成 CS1107：</span><span class="sxs-lookup"><span data-stu-id="ff8dc-107">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
