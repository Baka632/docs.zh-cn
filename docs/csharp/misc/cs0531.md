# <a name="compiler-error-cs0531"></a><span data-ttu-id="7f720-103">编译器错误 CS0531</span><span class="sxs-lookup"><span data-stu-id="7f720-103">Compiler Error CS0531</span></span>

<span data-ttu-id="7f720-104">“member”：接口成员不能有定义</span><span class="sxs-lookup"><span data-stu-id="7f720-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="7f720-105">在 [接口](../language-reference/keywords/interface.md) 中声明的方法必须在从接口继承的类而不是接口本身中实现。</span><span class="sxs-lookup"><span data-stu-id="7f720-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="7f720-106">下面的示例生成 CS0531：</span><span class="sxs-lookup"><span data-stu-id="7f720-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
