# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="56eb4-103">编译器警告（等级 2）CS0469</span><span class="sxs-lookup"><span data-stu-id="56eb4-103">Compiler Warning (level 2) CS0469</span></span>

<span data-ttu-id="56eb4-104">“goto case”值不可隐式转换为类型“type”</span><span class="sxs-lookup"><span data-stu-id="56eb4-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="56eb4-105">使用 `goto case`时，必须将 goto case 的值隐式转换为 switch 类型。</span><span class="sxs-lookup"><span data-stu-id="56eb4-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="56eb4-106">示例</span><span class="sxs-lookup"><span data-stu-id="56eb4-106">Example</span></span>  

 <span data-ttu-id="56eb4-107">下面的示例生成 CS0469。</span><span class="sxs-lookup"><span data-stu-id="56eb4-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
