# <a name="compiler-error-cs1620"></a><span data-ttu-id="d5d67-103">编译器错误 CS1620</span><span class="sxs-lookup"><span data-stu-id="d5d67-103">Compiler Error CS1620</span></span>
<span data-ttu-id="d5d67-104">参数“number”必须使用关键字“keyword”传递</span><span class="sxs-lookup"><span data-stu-id="d5d67-104">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="d5d67-105">如果将参数传递到采用 [ref](../language-reference/keywords/ref.md) 或 [out](../language-reference/keywords/out-parameter-modifier.md) 参数的函数，并且在调用时不包含 `ref` 或 `out` 关键字，或包括了错误的关键字，则都会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="d5d67-105">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="d5d67-106">错误文本指示要使用的相应关键字，以及导致失败的参数。</span><span class="sxs-lookup"><span data-stu-id="d5d67-106">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="d5d67-107">下面的示例生成 CS1620：</span><span class="sxs-lookup"><span data-stu-id="d5d67-107">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
