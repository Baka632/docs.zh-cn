# <a name="compiler-error-cs0534"></a><span data-ttu-id="8ac1b-103">编译器错误 CS0534</span><span class="sxs-lookup"><span data-stu-id="8ac1b-103">Compiler Error CS0534</span></span>

<span data-ttu-id="8ac1b-104">“function1”不实现继承的抽象成员“function2”</span><span class="sxs-lookup"><span data-stu-id="8ac1b-104">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="8ac1b-105">一个类需要实现基类中的所有 [抽象](../language-reference/keywords/abstract.md) 成员，除非该类也是抽象的。</span><span class="sxs-lookup"><span data-stu-id="8ac1b-105">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="8ac1b-106">下面的示例生成 CS0534：</span><span class="sxs-lookup"><span data-stu-id="8ac1b-106">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
