# <a name="compiler-error-cs1914"></a><span data-ttu-id="f2bd5-102">编译器错误 CS1914</span><span class="sxs-lookup"><span data-stu-id="f2bd5-102">Compiler Error CS1914</span></span>

<span data-ttu-id="f2bd5-103">无法在对象初始值设定项中为静态字段“name”赋值</span><span class="sxs-lookup"><span data-stu-id="f2bd5-103">Static field 'name' cannot be assigned in an object initializer</span></span>  
  
 <span data-ttu-id="f2bd5-104">根据定义，对象初始值设定项初始化类的对象或实例。</span><span class="sxs-lookup"><span data-stu-id="f2bd5-104">Object initializers by definition initialize objects, or instances, of classes.</span></span> <span data-ttu-id="f2bd5-105">它们不能用于初始化类型的 `static` 字段。</span><span class="sxs-lookup"><span data-stu-id="f2bd5-105">They cannot be used to initialize a `static` field of a type.</span></span> <span data-ttu-id="f2bd5-106">无论创建多少个类的实例， `static` 字段仅有一个副本。</span><span class="sxs-lookup"><span data-stu-id="f2bd5-106">No matter how many instances of a class are created, there is only one copy of a `static` field.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f2bd5-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f2bd5-107">To correct this error</span></span>  
  
1. <span data-ttu-id="f2bd5-108">将字段更改为类型中的实例字段，或放弃从对象初始值设定项中初始化它的尝试。</span><span class="sxs-lookup"><span data-stu-id="f2bd5-108">Either change the field to an instance field in the type, or remove the attempt to initialize it from the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f2bd5-109">示例</span><span class="sxs-lookup"><span data-stu-id="f2bd5-109">Example</span></span>  

 <span data-ttu-id="f2bd5-110">以下代码生成 CS1914，因为初始值设定项尝试初始化为 `TestClass.Number` 的 `static`字段：</span><span class="sxs-lookup"><span data-stu-id="f2bd5-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
