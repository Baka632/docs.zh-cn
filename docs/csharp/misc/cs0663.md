# <a name="compiler-error-cs0663"></a><span data-ttu-id="954ed-103">编译器错误 CS0663</span><span class="sxs-lookup"><span data-stu-id="954ed-103">Compiler Error CS0663</span></span>
<span data-ttu-id="954ed-104">无法定义仅在 ref 和 out 上存在不同的重载方法。</span><span class="sxs-lookup"><span data-stu-id="954ed-104">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="954ed-105">不允许仅在参数上使用、 [ref](../language-reference/keywords/ref.md)和[out](../language-reference/keywords/out-parameter-modifier.md)等[不同的方法](../language-reference/keywords/in-parameter-modifier.md)。</span><span class="sxs-lookup"><span data-stu-id="954ed-105">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="954ed-106">以下示例生成 CS0663：</span><span class="sxs-lookup"><span data-stu-id="954ed-106">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
