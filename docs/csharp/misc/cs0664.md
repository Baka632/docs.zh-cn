 <span data-ttu-id="cc22c-105">赋值无法完成，请使用后缀更正指令。</span><span class="sxs-lookup"><span data-stu-id="cc22c-105">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="cc22c-106">每个类型的文档确定该类型的相应后缀。</span><span class="sxs-lookup"><span data-stu-id="cc22c-106">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="cc22c-107">有关转换的详细信息，请参阅 [强制转换和类型转换](../programming-guide/types/casting-and-type-conversions.md)。</span><span class="sxs-lookup"><span data-stu-id="cc22c-107">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="cc22c-108">下面的示例生成 CS0664：</span><span class="sxs-lookup"><span data-stu-id="cc22c-108">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
