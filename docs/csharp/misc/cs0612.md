# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="eb542-103">编译器警告 (等级 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="eb542-103">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="eb542-104">“member” 已过时</span><span class="sxs-lookup"><span data-stu-id="eb542-104">'member' is obsolete</span></span>  
  
 <span data-ttu-id="eb542-105">类设计器将具有 [过时特性](../language-reference/attributes/general.md#obsolete-attribute)的成员标记为。</span><span class="sxs-lookup"><span data-stu-id="eb542-105">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="eb542-106">这意味着该类的未来版本中可能不支持该成员。</span><span class="sxs-lookup"><span data-stu-id="eb542-106">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="eb542-107">下面的示例显示访问过时的成员如何生成 CS0612：</span><span class="sxs-lookup"><span data-stu-id="eb542-107">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
