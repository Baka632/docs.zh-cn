 <span data-ttu-id="150d1-107">下面的示例生成 CS0205：</span><span class="sxs-lookup"><span data-stu-id="150d1-107">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass

   abstract public void M();  
}

{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
