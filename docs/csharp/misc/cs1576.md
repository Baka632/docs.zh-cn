# <a name="compiler-error-cs1576"></a><span data-ttu-id="64ea6-103">编译器错误 CS1576</span><span class="sxs-lookup"><span data-stu-id="64ea6-103">Compiler Error CS1576</span></span>

 <span data-ttu-id="64ea6-105">编译器检测到传递给 [#line](../language-reference/preprocessor-directives/preprocessor-line.md) 指令的值有错误。</span><span class="sxs-lookup"><span data-stu-id="64ea6-105">The compiler detected an error with the value passed to the [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>

```csharp

   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead

      intt i;  // error will be reported on line 101  
   }  
}  
```
