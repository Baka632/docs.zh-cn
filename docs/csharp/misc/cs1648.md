## <a name="example"></a><span data-ttu-id="aaa1f-107">示例</span><span class="sxs-lookup"><span data-stu-id="aaa1f-107">Example</span></span>

 <span data-ttu-id="aaa1f-108">下面的示例生成 CS1648：</span><span class="sxs-lookup"><span data-stu-id="aaa1f-108">The following sample generates CS1648:</span></span>

```csharp
// CS1648.cs
public struct Inner
{
    public int i;
}

class Outer
{
    public readonly Inner inner = new Inner();
}

class D
{
    static void Main()
    {
        var outer = new Outer();
        outer.inner.i = 1;  // CS1648
    }
}
```
