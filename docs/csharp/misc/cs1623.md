# <a name="compiler-error-cs1623"></a><span data-ttu-id="8a458-103">编译器错误 CS1623</span><span class="sxs-lookup"><span data-stu-id="8a458-103">Compiler Error CS1623</span></span>
<span data-ttu-id="8a458-104">迭代器不能有 ref 或 out 参数</span><span class="sxs-lookup"><span data-stu-id="8a458-104">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="8a458-105">如果迭代器方法采用 `in` 、或参数，则会发生此错误 `ref` `out` 。</span><span class="sxs-lookup"><span data-stu-id="8a458-105">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="8a458-106">若要避免此错误，请 `in` `ref` `out` 从方法签名中删除、或关键字。</span><span class="sxs-lookup"><span data-stu-id="8a458-106">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8a458-107">示例</span><span class="sxs-lookup"><span data-stu-id="8a458-107">Example</span></span>  
 <span data-ttu-id="8a458-108">以下示例生成 CS1623：</span><span class="sxs-lookup"><span data-stu-id="8a458-108">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
