# <a name="compiler-error-cs0736"></a><span data-ttu-id="3dc90-103">编译器错误 CS0736</span><span class="sxs-lookup"><span data-stu-id="3dc90-103">Compiler Error CS0736</span></span>
<span data-ttu-id="3dc90-104">“type name”不实现接口成员“member name”。</span><span class="sxs-lookup"><span data-stu-id="3dc90-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="3dc90-105">“method name”无法实现接口成员，因为它是静态的。</span><span class="sxs-lookup"><span data-stu-id="3dc90-105">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="3dc90-106">在将静态方法隐式或显式声明为接口成员的实现时，会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="3dc90-106">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3dc90-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="3dc90-107">To correct this error</span></span>  
  
- <span data-ttu-id="3dc90-108">从方法声明中删除 [static](../language-reference/keywords/static.md) 修饰符。</span><span class="sxs-lookup"><span data-stu-id="3dc90-108">Remove the [static](../language-reference/keywords/static.md) modifier from the method declaration.</span></span>  
  
- <span data-ttu-id="3dc90-109">更改接口方法的名称。</span><span class="sxs-lookup"><span data-stu-id="3dc90-109">Change the name of the interface method.</span></span>  
  
- <span data-ttu-id="3dc90-110">重新定义包含类型，使其不从接口继承。</span><span class="sxs-lookup"><span data-stu-id="3dc90-110">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3dc90-111">示例</span><span class="sxs-lookup"><span data-stu-id="3dc90-111">Example</span></span>  
 <span data-ttu-id="3dc90-112">下面的代码生成 CS0736，因为 `Program.testMethod` 被声明为静态的：</span><span class="sxs-lookup"><span data-stu-id="3dc90-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
