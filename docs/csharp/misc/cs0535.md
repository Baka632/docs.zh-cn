 <span data-ttu-id="3ee77-105">[类](../language-reference/keywords/class.md) 派生自 [接口](../language-reference/keywords/interface.md)，但该类未实现一个或多个接口的成员。</span><span class="sxs-lookup"><span data-stu-id="3ee77-105">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="3ee77-106">类必须实现其派生自的接口的所有成员，或者被声明为 `abstract`。</span><span class="sxs-lookup"><span data-stu-id="3ee77-106">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3ee77-107">示例</span><span class="sxs-lookup"><span data-stu-id="3ee77-107">Example</span></span>  
 <span data-ttu-id="3ee77-108">下面的示例生成 CS0535。</span><span class="sxs-lookup"><span data-stu-id="3ee77-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
