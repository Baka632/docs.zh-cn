# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="b759f-103">编译器警告（等级 3）CS0659</span><span class="sxs-lookup"><span data-stu-id="b759f-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="b759f-104">“class”重写 Object.Equals(object o) 但不重写 Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="b759f-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="b759f-105">编译器检测到对方法的重写， <xref:System.Object.Equals%2A?displayProperty=nameWithType> 但不会重写 <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> 方法。</span><span class="sxs-lookup"><span data-stu-id="b759f-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="b759f-106">重写 <xref:System.Object.Equals%2A> 意味着还需要重写 <xref:System.Object.GetHashCode%2A> 。</span><span class="sxs-lookup"><span data-stu-id="b759f-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="b759f-107">下面的代码生成 CS0659：</span><span class="sxs-lookup"><span data-stu-id="b759f-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
