# <a name="compiler-error-cs0738"></a><span data-ttu-id="33b12-103">编译器错误 CS0738</span><span class="sxs-lookup"><span data-stu-id="33b12-103">Compiler Error CS0738</span></span>
<span data-ttu-id="33b12-104">“type name”不实现接口成员“member name”。</span><span class="sxs-lookup"><span data-stu-id="33b12-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="33b12-105">“method name”不能实现“interface member”，因为它没有“type name”的匹配返回类型。</span><span class="sxs-lookup"><span data-stu-id="33b12-105">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="33b12-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="33b12-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="33b12-109">示例</span><span class="sxs-lookup"><span data-stu-id="33b12-109">Example</span></span>  
 <span data-ttu-id="33b12-110">下面的代码生成 CS0738，因为类方法将返回 `void` ，同名的接口成员返回 `int`：</span><span class="sxs-lookup"><span data-stu-id="33b12-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
