# <a name="compiler-error-cs0065"></a><span data-ttu-id="b51cc-103">编译器错误 CS0065</span><span class="sxs-lookup"><span data-stu-id="b51cc-103">Compiler Error CS0065</span></span>

<span data-ttu-id="b51cc-104">“event”：事件属性必须同时具有 add 和 remove 访问器</span><span class="sxs-lookup"><span data-stu-id="b51cc-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="b51cc-105">并不是字段的事件必须有两种访问方法。</span><span class="sxs-lookup"><span data-stu-id="b51cc-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="b51cc-106">以下示例生成 CS0065：</span><span class="sxs-lookup"><span data-stu-id="b51cc-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
