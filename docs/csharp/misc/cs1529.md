## <a name="example"></a><span data-ttu-id="b9dd7-106">示例</span><span class="sxs-lookup"><span data-stu-id="b9dd7-106">Example</span></span>

```csharp

    namespace Subspace  
    {  
        using Microsoft;

}

```
