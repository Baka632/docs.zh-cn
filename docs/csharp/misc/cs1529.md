## <a name="example"></a><span data-ttu-id="487c6-106">示例</span><span class="sxs-lookup"><span data-stu-id="487c6-106">Example</span></span>  

 <span data-ttu-id="487c6-107">下面的示例生成 CS1529:</span><span class="sxs-lookup"><span data-stu-id="487c6-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
