# <a name="compiler-error-cs0538"></a><span data-ttu-id="80f02-103">编译器错误 CS0538</span><span class="sxs-lookup"><span data-stu-id="80f02-103">Compiler Error CS0538</span></span>

<span data-ttu-id="80f02-104">显式接口声明中的“name”不是接口</span><span class="sxs-lookup"><span data-stu-id="80f02-104">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="80f02-105">尝试显式声明 [接口](../language-reference/keywords/interface.md)，但未指定接口。</span><span class="sxs-lookup"><span data-stu-id="80f02-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="80f02-106">下面的示例生成 CS0538：</span><span class="sxs-lookup"><span data-stu-id="80f02-106">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
