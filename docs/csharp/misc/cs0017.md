<span data-ttu-id="3f454-104">程序“输出文档名称”定义了多个入口点。</span><span class="sxs-lookup"><span data-stu-id="3f454-104">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="3f454-105">请使用 /main 进行编译，以指定包含入口点的类型。</span><span class="sxs-lookup"><span data-stu-id="3f454-105">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="3f454-106">一个程序只能有一个 [Main](../programming-guide/main-and-command-args/index.md) 方法。</span><span class="sxs-lookup"><span data-stu-id="3f454-106">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="3f454-107">若要解决此错误，可以删除代码中的所有 Main 方法（除了一个方法除外），也可以使用 [-main](../language-reference/compiler-options/main-compiler-option.md) 编译器选项来指定想要使用的 main 方法。</span><span class="sxs-lookup"><span data-stu-id="3f454-107">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>

```csharp  
{  
   static public void Main()

   }

   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }

```
