## <a name="example"></a><span data-ttu-id="ff470-108">示例</span><span class="sxs-lookup"><span data-stu-id="ff470-108">Example</span></span>  

 <span data-ttu-id="ff470-109">下面的示例生成 CS0026：</span><span class="sxs-lookup"><span data-stu-id="ff470-109">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
