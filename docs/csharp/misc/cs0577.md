# <a name="compiler-error-cs0577"></a><span data-ttu-id="6ccff-103">编译器错误 CS0577</span><span class="sxs-lookup"><span data-stu-id="6ccff-103">Compiler Error CS0577</span></span>

<span data-ttu-id="6ccff-104">Conditional 特性对“function”无效，因为它是构造函数、析构函数、运算符或显式接口实现</span><span class="sxs-lookup"><span data-stu-id="6ccff-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="6ccff-105">`Conditional` 无法应用于指定的方法。</span><span class="sxs-lookup"><span data-stu-id="6ccff-105">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="6ccff-106">例如，你无法对显式接口定义使用某些特性。</span><span class="sxs-lookup"><span data-stu-id="6ccff-106">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="6ccff-107">下面的示例生成 CS0577：</span><span class="sxs-lookup"><span data-stu-id="6ccff-107">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
