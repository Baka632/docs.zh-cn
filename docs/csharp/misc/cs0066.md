# <a name="compiler-error-cs0066"></a><span data-ttu-id="52696-103">编译器错误 CS0066</span><span class="sxs-lookup"><span data-stu-id="52696-103">Compiler Error CS0066</span></span>
<span data-ttu-id="52696-104">“event”: 事件必须是委托类型</span><span class="sxs-lookup"><span data-stu-id="52696-104">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="52696-105">事件关键字需要 [委托](../language-reference/builtin-types/reference-types.md) 类型。</span><span class="sxs-lookup"><span data-stu-id="52696-105">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="52696-106">有关详细信息，请参阅 [事件](../programming-guide/events/index.md) 和 [委托](../programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="52696-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="52696-107">下面的示例生成 CS0066：</span><span class="sxs-lookup"><span data-stu-id="52696-107">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
