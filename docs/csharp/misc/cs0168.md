# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="2f542-103">编译器警告（等级 3）CS0168</span><span class="sxs-lookup"><span data-stu-id="2f542-103">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="2f542-104">变量 "var" 已声明，但从未使用过</span><span class="sxs-lookup"><span data-stu-id="2f542-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="2f542-105">当你声明一个变量但不使用它时，编译器会发出一个3级警告。</span><span class="sxs-lookup"><span data-stu-id="2f542-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="2f542-106">下面的示例生成一个 CS0168 警告：</span><span class="sxs-lookup"><span data-stu-id="2f542-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
