# <a name="compiler-error-cs0405"></a>编译器错误 CS0405
类型参数“type parameter”的重复约束“constraint”  
  
 泛型声明的两个约束完全相同。 若要消除此错误，请删除重复的约束。  
  
 下面的示例生成 CS0405：  
  
```csharp  
// CS0405.cs  
interface I  
{  
}  
  
class C<T> where T : I, I  // CS0405.cs  
{  
}  
```
