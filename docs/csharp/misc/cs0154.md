## <a name="example"></a><span data-ttu-id="559e2-107">示例</span><span class="sxs-lookup"><span data-stu-id="559e2-107">Example</span></span>  
 <span data-ttu-id="559e2-108">下面的示例生成 CS0154：</span><span class="sxs-lookup"><span data-stu-id="559e2-108">The following sample generates CS0154:</span></span>  
  
```csharp  
// CS0154.cs  
public class MyClass2  
{  
    public int i  
    {  
        set  
        {  
        }  
        // uncomment the get method to resolve this error  
        /*  
        get  
        {  
            return 0;  
        }  
        */  
    }  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClass2 myClass2 = new MyClass2();  
        int j = myClass2.i;   // CS0154, no get method  
    }  
}  
```
