# <a name="compiler-error-cs0690"></a>编译器错误 CS0690
输入文件“file”包含无效的元数据。  
  
 可打开元数据文件，但由于某些本地化的问题，元数据已损坏。 这与错误 [CS0009](./cs0009.md)相似，只不过可以打开元数据文件。  
  
 PE（“进程可执行文件”）是一个可执行文件。
