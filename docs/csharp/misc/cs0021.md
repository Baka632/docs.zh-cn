## <a name="example"></a><span data-ttu-id="40219-109">示例</span><span class="sxs-lookup"><span data-stu-id="40219-109">Example</span></span>

<span data-ttu-id="40219-110">下面的 c + + 示例将编译为 .dll 文件。</span><span class="sxs-lookup"><span data-stu-id="40219-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="40219-111">请注意， [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) 属性会被注释掉，以生成错误。</span><span class="sxs-lookup"><span data-stu-id="40219-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="40219-112">下面的 c # 示例将调用 .dll 文件。</span><span class="sxs-lookup"><span data-stu-id="40219-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="40219-113">它尝试通过索引器访问类，但由于没有将任何成员声明为默认索引器，因此会生成错误。</span><span class="sxs-lookup"><span data-stu-id="40219-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="40219-114">可以通过取消注释行来解决该错误 `[DefaultMember("myItem")]` 。</span><span class="sxs-lookup"><span data-stu-id="40219-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp
// CS0021.cs
// compile with: /reference:CPP0021.dll
public class MyClass
{
    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();
        int j = myMC[1]; // CS0021
    }
}
```
