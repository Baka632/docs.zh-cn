# <a name="compiler-error-cs0104"></a><span data-ttu-id="eb3e1-103">编译器错误 CS0104</span><span class="sxs-lookup"><span data-stu-id="eb3e1-103">Compiler Error CS0104</span></span>
<span data-ttu-id="eb3e1-104">“reference”是“identifier”和“identifier”之间的不明确引用</span><span class="sxs-lookup"><span data-stu-id="eb3e1-104">'reference' is an ambiguous reference between 'identifier' and 'identifier'</span></span>  
  
 <span data-ttu-id="eb3e1-105">程序包含两个命名空间的 [使用](../language-reference/keywords/using.md) 指令，而代码引用了在两个命名空间中都出现的名称。</span><span class="sxs-lookup"><span data-stu-id="eb3e1-105">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>  
  
 <span data-ttu-id="eb3e1-106">以下示例生成 CS0104：</span><span class="sxs-lookup"><span data-stu-id="eb3e1-106">The following sample generates CS0104:</span></span>  
  
```csharp  
// CS0104.cs  
using x;  
using y;  
  
namespace x  
{  
   public class Test  
   {  
   }  
}  
  
namespace y  
{  
   public class Test  
   {  
   }  
}  
  
public class a  
{  
   public static void Main()  
   {  
      Test test = new Test();   // CS0104, is Test in x or y namespace?  
      // try the following line instead  
      // y.Test test = new y.Test();  
   }  
}  
```
