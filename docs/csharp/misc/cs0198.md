# <a name="compiler-error-cs0198"></a><span data-ttu-id="b27a7-103">编译器错误 CS0198</span><span class="sxs-lookup"><span data-stu-id="b27a7-103">Compiler Error CS0198</span></span>

<span data-ttu-id="b27a7-104">无法给静态只读字段“name”的字段赋值（静态构造函数或变量初始值设定项中除外）</span><span class="sxs-lookup"><span data-stu-id="b27a7-104">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="b27a7-105">[readonly](../language-reference/keywords/readonly.md) 变量必须与构造函数具有相同的 [静态](../language-reference/keywords/static.md) 用法，在此构造函数中你要对其进行初始化。</span><span class="sxs-lookup"><span data-stu-id="b27a7-105">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="b27a7-106">有关详细信息，请参阅[静态构造函数](../programming-guide/classes-and-structs/static-constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="b27a7-106">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>

```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
