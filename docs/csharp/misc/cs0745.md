# <a name="compiler-error-cs0745"></a><span data-ttu-id="e8462-103">编译器错误 CS0745</span><span class="sxs-lookup"><span data-stu-id="e8462-103">Compiler Error CS0745</span></span>

<span data-ttu-id="e8462-104">应为上下文关键字 "by"</span><span class="sxs-lookup"><span data-stu-id="e8462-104">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="e8462-105">`group` 子句的模式是 `group...by` 跟一个可选 `into`，如以下示例所示：</span><span class="sxs-lookup"><span data-stu-id="e8462-105">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

var query = from name in names  
            group name by name[0];  
```

             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e8462-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="e8462-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="e8462-109">示例</span><span class="sxs-lookup"><span data-stu-id="e8462-109">Example</span></span>  

 <span data-ttu-id="e8462-110">以下代码生成 CS0745：</span><span class="sxs-lookup"><span data-stu-id="e8462-110">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
