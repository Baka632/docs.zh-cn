## <a name="example"></a><span data-ttu-id="935da-106">示例</span><span class="sxs-lookup"><span data-stu-id="935da-106">Example</span></span>  
 <span data-ttu-id="935da-107">以下示例生成 CS0448。</span><span class="sxs-lookup"><span data-stu-id="935da-107">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```
