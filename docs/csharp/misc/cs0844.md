## <a name="example"></a><span data-ttu-id="32153-112">示例</span><span class="sxs-lookup"><span data-stu-id="32153-112">Example</span></span>  
 <span data-ttu-id="32153-113">下面的代码生成 CS0844：</span><span class="sxs-lookup"><span data-stu-id="32153-113">The following code generates CS0844:</span></span>  
  
```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="32153-114">更正错误，方法是使用 `this.num` 以引用类字段</span><span class="sxs-lookup"><span data-stu-id="32153-114">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

```csharp  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // Error fixed.
            int num2 = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
