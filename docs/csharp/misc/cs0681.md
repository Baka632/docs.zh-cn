# <a name="compiler-error-cs0681"></a>编译器错误 CS0681
修饰符“abstract”对于字段无效。 请尝试改用属性。  
  
 不能使字段为抽象。 但是，可以有访问该字段的抽象属性。  
  
## <a name="example"></a>示例  
 以下示例生成 CS0681：  
  
```csharp  
// CS0681.cs  
// compile with: /target:library  
abstract class C  
{  
    abstract int num;  // CS0681  
}  
```  
  
## <a name="example"></a>示例  
 改为尝试使用以下代码：  
  
```csharp  
// CS0681b.cs  
// compile with: /target:library  
abstract class C  
{  
    public abstract int num  
    {  
       get;  
       set;  
    }  
}  
```
