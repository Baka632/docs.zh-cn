# <a name="compiler-error-cs1554"></a><span data-ttu-id="407b1-103">编译器错误 CS1554</span><span class="sxs-lookup"><span data-stu-id="407b1-103">Compiler Error CS1554</span></span>
<span data-ttu-id="407b1-104">声明无效;使用 " \<type> operator op ( ..."是</span><span class="sxs-lookup"><span data-stu-id="407b1-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="407b1-105">[重载运算符](../language-reference/operators/operator-overloading.md)的返回类型必须出现在 `operator` 关键字之前。</span><span class="sxs-lookup"><span data-stu-id="407b1-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>

```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
