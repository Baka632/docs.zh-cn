# <a name="compiler-error-cs0750"></a><span data-ttu-id="367e9-103">编译器错误 CS0750</span><span class="sxs-lookup"><span data-stu-id="367e9-103">Compiler Error CS0750</span></span>
<span data-ttu-id="367e9-104">分部方法不能具有访问修饰符或 virtual、abstract、override、new、sealed 或 extern 修饰符。</span><span class="sxs-lookup"><span data-stu-id="367e9-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="367e9-105">由于其特殊行为，分部方法需遵守其可接受的修饰符相关限制。</span><span class="sxs-lookup"><span data-stu-id="367e9-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="367e9-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="367e9-106">To correct this error</span></span>  
  
1. <span data-ttu-id="367e9-107">删除方法声明中未经授权的修饰符。</span><span class="sxs-lookup"><span data-stu-id="367e9-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="367e9-108">示例</span><span class="sxs-lookup"><span data-stu-id="367e9-108">Example</span></span>  
 <span data-ttu-id="367e9-109">下面的示例生成 CS0750：</span><span class="sxs-lookup"><span data-stu-id="367e9-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
