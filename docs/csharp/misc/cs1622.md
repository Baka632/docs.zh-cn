 <span data-ttu-id="b501a-108">下面的示例生成 CS1622：</span><span class="sxs-lookup"><span data-stu-id="b501a-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library

class C : IEnumerable

   public IEnumerator GetEnumerator()

      return (IEnumerator) this;  // CS1622

   }  
}  
```
