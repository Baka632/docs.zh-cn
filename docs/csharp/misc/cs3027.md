# <a name="compiler-warning-level-1-cs3027"></a><span data-ttu-id="c77ec-102">编译器警告（等级 1）CS3027</span><span class="sxs-lookup"><span data-stu-id="c77ec-102">Compiler Warning (level 1) CS3027</span></span>

<span data-ttu-id="c77ec-103">“type_1”类型不符合 CLS，因为基接口“type_2”不符合 CLS</span><span class="sxs-lookup"><span data-stu-id="c77ec-103">'type_1' is not CLS-compliant because base interface 'type_2' is not CLS-compliant</span></span>

## <a name="example"></a><span data-ttu-id="c77ec-105">示例</span><span class="sxs-lookup"><span data-stu-id="c77ec-105">Example</span></span>  

 <span data-ttu-id="c77ec-106">下面的示例包含的接口中有一个方法，该方法在其签名中使用不符合 CLS 的类型，致使类型不符合 CLS。</span><span class="sxs-lookup"><span data-stu-id="c77ec-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>  
  
```csharp  
// CS3027.cs  
// compile with: /target:library  
public interface IBase  
{  
   void IMethod(uint i);  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c77ec-107">示例</span><span class="sxs-lookup"><span data-stu-id="c77ec-107">Example</span></span>  

 <span data-ttu-id="c77ec-108">下面的示例生成 CS3027。</span><span class="sxs-lookup"><span data-stu-id="c77ec-108">The following sample generates CS3027.</span></span>

```csharp  
// CS3027_b.cs

```
