## <a name="example"></a><span data-ttu-id="94dd0-107">示例</span><span class="sxs-lookup"><span data-stu-id="94dd0-107">Example</span></span>

```csharp  
      j++;

   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error

   }  
}  
```
