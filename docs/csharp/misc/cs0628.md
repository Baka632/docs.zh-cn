# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="f3771-103">编译器警告（等级 4）CS0628</span><span class="sxs-lookup"><span data-stu-id="f3771-103">Compiler Warning (level 4) CS0628</span></span>

 <span data-ttu-id="f3771-105">[密封](../language-reference/keywords/sealed.md) 类不能引入 [保护](../language-reference/keywords/protected.md) 成员，因为其他类不能继承 `sealed` 类并使用 `protected` 成员。</span><span class="sxs-lookup"><span data-stu-id="f3771-105">A [sealed](../language-reference/keywords/sealed.md) class cannot introduce a [protected](../language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>

```csharp  
{  
   protected int i;   // CS0628  
}

{  
   public static void Main()  
   {  
   }  
}  
```
