# <a name="compiler-error-cs0698"></a>编译器错误 CS0698
无法从“类”派生泛型类型，因为它是一个特性类  
  
 从特性类中派生的任何类都是特性。 特性不能是泛型类型。  
  
 下面的示例生成 CS0698:  
  
```csharp  
// CS0698.cs  
class C<T> : System.Attribute  // CS0698  
{  
}  
```
