 <span data-ttu-id="bf22a-105">标签在 [switch](../language-reference/keywords/switch.md) 语句中重复。</span><span class="sxs-lookup"><span data-stu-id="bf22a-105">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="bf22a-106">有关更多信息，请参见 [switch](../language-reference/keywords/switch.md)。</span><span class="sxs-lookup"><span data-stu-id="bf22a-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="bf22a-107">以下示例生成 CS0152：</span><span class="sxs-lookup"><span data-stu-id="bf22a-107">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
