 <span data-ttu-id="47230-106">C# 编译器不允许使用 <xref:System.ParamArrayAttribute?displayProperty=nameWithType>；请改用 [杂注](../language-reference/keywords/params.md) 。</span><span class="sxs-lookup"><span data-stu-id="47230-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>

```csharp  
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674

   }  
  
   public static void Main()

      UseParams(1, 2, 3);  
   }  
}  
```
