# <a name="compiler-error-cs0238"></a><span data-ttu-id="e2750-103">编译器错误 CS0238</span><span class="sxs-lookup"><span data-stu-id="e2750-103">Compiler Error CS0238</span></span>
<span data-ttu-id="e2750-104">因为“member”不是重写，所以无法将其密封</span><span class="sxs-lookup"><span data-stu-id="e2750-104">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="e2750-105">在尚未标记为[override](../language-reference/keywords/override.md)的成员上使用了[sealed](../language-reference/keywords/sealed.md) 。</span><span class="sxs-lookup"><span data-stu-id="e2750-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="e2750-106">有关详细信息，请参阅[继承](../programming-guide/classes-and-structs/inheritance.md)。</span><span class="sxs-lookup"><span data-stu-id="e2750-106">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="e2750-107">下面的示例生成 CS0238：</span><span class="sxs-lookup"><span data-stu-id="e2750-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
