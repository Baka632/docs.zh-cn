# <a name="compiler-error-cs1647"></a>编译器错误 CS1647

表达式太长或者过于复杂，无法在“code”附近编译  
  
 处理你的代码的编译器中有堆栈溢出。 若要解决此错误，请简化你的代码。 如果你的代码有效，请在 [开发人员社区](https://developercommunity.visualstudio.com/spaces/61/index.html) 网站或 [StackOverflow](https://stackoverflow.com/)上获取帮助。
