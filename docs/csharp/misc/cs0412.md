# <a name="compiler-error-cs0412"></a><span data-ttu-id="869e5-103">编译器错误 CS0412</span><span class="sxs-lookup"><span data-stu-id="869e5-103">Compiler Error CS0412</span></span>

<span data-ttu-id="869e5-104">“generic”：参数或局部变量不能与方法类型参数同名</span><span class="sxs-lookup"><span data-stu-id="869e5-104">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="869e5-105">泛型方法的类型参数和此方法的局部变量或其中一项方法的参数之间存在名称冲突。</span><span class="sxs-lookup"><span data-stu-id="869e5-105">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="869e5-106">若要避免此错误，重命名任何存在冲突的参数或局部变量。</span><span class="sxs-lookup"><span data-stu-id="869e5-106">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="869e5-107">示例</span><span class="sxs-lookup"><span data-stu-id="869e5-107">Example</span></span>  

 <span data-ttu-id="869e5-108">以下示例生成 CS0412：</span><span class="sxs-lookup"><span data-stu-id="869e5-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
