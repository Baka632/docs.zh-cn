# <a name="compiler-error-cs0110"></a><span data-ttu-id="e6a64-103">编译器错误 CS0110</span><span class="sxs-lookup"><span data-stu-id="e6a64-103">Compiler Error CS0110</span></span>
<span data-ttu-id="e6a64-104">“const declaration”的常量值计算涉及循环定义</span><span class="sxs-lookup"><span data-stu-id="e6a64-104">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="e6a64-105">[const](../language-reference/keywords/const.md) 变量 (`a`) 的声明不能引用另一个也引用 (`b`) 的 const 变量 (`a`)。</span><span class="sxs-lookup"><span data-stu-id="e6a64-105">The declaration of a [const](../language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="e6a64-106">下面的示例生成 CS0110：</span><span class="sxs-lookup"><span data-stu-id="e6a64-106">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
