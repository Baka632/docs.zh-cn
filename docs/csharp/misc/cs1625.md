# <a name="compiler-error-cs1625"></a><span data-ttu-id="1f399-103">编译器错误 CS1625</span><span class="sxs-lookup"><span data-stu-id="1f399-103">Compiler Error CS1625</span></span>

<span data-ttu-id="1f399-104">无法在 finally 子句体中生成</span><span class="sxs-lookup"><span data-stu-id="1f399-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="1f399-105">在 finally 子句体内不允许出现 yield 语句。</span><span class="sxs-lookup"><span data-stu-id="1f399-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="1f399-106">若要避免此错误，请将 yield 语句移到 finally 子句的外部。</span><span class="sxs-lookup"><span data-stu-id="1f399-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="1f399-107">下面的示例生成 CS1625：</span><span class="sxs-lookup"><span data-stu-id="1f399-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
