 <span data-ttu-id="5d8e5-105">使用 `override`、 **virtual**或 **abstract** 关键字的任何方法声明也不能使用 **static** 关键字。</span><span class="sxs-lookup"><span data-stu-id="5d8e5-105">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>

 <span data-ttu-id="5d8e5-107">下面的示例生成 CS0112:</span><span class="sxs-lookup"><span data-stu-id="5d8e5-107">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
