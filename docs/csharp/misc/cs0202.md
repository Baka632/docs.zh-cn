# <a name="compiler-error-cs0202"></a><span data-ttu-id="41a9d-103">编译器错误 CS0202</span><span class="sxs-lookup"><span data-stu-id="41a9d-103">Compiler Error CS0202</span></span>

<span data-ttu-id="41a9d-104">foreach 要求 "type.GetEnumerator()" 的返回类型“type”必须具有适当的公共 MoveNext 方法和公共 Current 属性</span><span class="sxs-lookup"><span data-stu-id="41a9d-104">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="41a9d-105">用于启用 foreach 语句的函数 <xref:System.Collections.IEnumerable.GetEnumerator%2A> 无法返回指针或数组；它必须返回能够充当枚举器的类的实例。</span><span class="sxs-lookup"><span data-stu-id="41a9d-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="41a9d-106">充当枚举器的适当要求包括一个公共 Current 属性和一个公共 MoveNext 方法。</span><span class="sxs-lookup"><span data-stu-id="41a9d-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>

```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
