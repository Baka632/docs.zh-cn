# <a name="compiler-error-cs1613"></a><span data-ttu-id="ff071-103">编译器错误 CS1613</span><span class="sxs-lookup"><span data-stu-id="ff071-103">Compiler Error CS1613</span></span>
<span data-ttu-id="ff071-104">找不到用于接口 “interface” coclass 包装类 “class” （是否缺少程序集引用？）</span><span class="sxs-lookup"><span data-stu-id="ff071-104">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="ff071-105">已尝试实例化来自接口的 COM 对象。</span><span class="sxs-lookup"><span data-stu-id="ff071-105">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="ff071-106">此接口具有 **ComImport** 和 `CoClass` 特性，但编译器找不到为 `CoClass` 特性给定的类型。</span><span class="sxs-lookup"><span data-stu-id="ff071-106">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="ff071-107">要解决此错误，你可以尝试以下项之一：</span><span class="sxs-lookup"><span data-stu-id="ff071-107">To resolve this error, you can try one of the following:</span></span>  
  
- <span data-ttu-id="ff071-108">向具有 coclass 的程序集添加一个引用（大多数情况下，接口和 coclass 应位于同一程序集中）</span><span class="sxs-lookup"><span data-stu-id="ff071-108">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="ff071-109">有关信息，请参阅 [-引用](../language-reference/compiler-options/reference-compiler-option.md) 或 [添加引用对话框](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) 。</span><span class="sxs-lookup"><span data-stu-id="ff071-109">See [-reference](../language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) for information.</span></span>  
  
- <span data-ttu-id="ff071-110">修复接口上的 `CoClass` 特性。</span><span class="sxs-lookup"><span data-stu-id="ff071-110">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="ff071-111">以下示例演示如何正确使用 **CoClassAttribute**：</span><span class="sxs-lookup"><span data-stu-id="ff071-111">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
