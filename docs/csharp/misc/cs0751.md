# <a name="compiler-error-cs0751"></a><span data-ttu-id="454ee-103">编译器错误 CS0751</span><span class="sxs-lookup"><span data-stu-id="454ee-103">Compiler Error CS0751</span></span>

<span data-ttu-id="454ee-104">分部方法必须在分部类或分部结构内声明</span><span class="sxs-lookup"><span data-stu-id="454ee-104">A partial method must be declared in a partial class or partial struct</span></span>  
  
 <span data-ttu-id="454ee-105">除非 [partial](../language-reference/keywords/partial-method.md) 方法封装在分部类或分部结构内，否则不能对其进行声明。</span><span class="sxs-lookup"><span data-stu-id="454ee-105">It is not possible to declare a [partial](../language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>
