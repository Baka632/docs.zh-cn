# <a name="compiler-error-cs0243"></a><span data-ttu-id="c8f55-103">编译器错误 CS0243</span><span class="sxs-lookup"><span data-stu-id="c8f55-103">Compiler error CS0243</span></span>

<span data-ttu-id="c8f55-104">Conditional 特性在“method”上无效，因为它是重写方法</span><span class="sxs-lookup"><span data-stu-id="c8f55-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="c8f55-105">使用 <xref:System.Diagnostics.ConditionalAttribute> [override](../language-reference/keywords/override.md) 关键字标记的方法上不允许使用特性。</span><span class="sxs-lookup"><span data-stu-id="c8f55-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="c8f55-106">有关详细信息，请参阅 [了解何时使用 Override 和 New 关键字](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)。</span><span class="sxs-lookup"><span data-stu-id="c8f55-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="c8f55-107">编译器从不绑定到重写方法。</span><span class="sxs-lookup"><span data-stu-id="c8f55-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="c8f55-108">它只绑定到基方法，公共语言运行时根据需要调用重写。</span><span class="sxs-lookup"><span data-stu-id="c8f55-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="c8f55-109">下面的代码生成 CS0243：</span><span class="sxs-lookup"><span data-stu-id="c8f55-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
