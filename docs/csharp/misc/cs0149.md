# <a name="compiler-error-cs0149"></a><span data-ttu-id="1f3ec-103">编译器错误 CS0149</span><span class="sxs-lookup"><span data-stu-id="1f3ec-103">Compiler Error CS0149</span></span>
<span data-ttu-id="1f3ec-104">应输入方法名称</span><span class="sxs-lookup"><span data-stu-id="1f3ec-104">Method name expected</span></span>  
  
 <span data-ttu-id="1f3ec-105">在创建 [委托](../language-reference/builtin-types/reference-types.md)时，指定一个方法。</span><span class="sxs-lookup"><span data-stu-id="1f3ec-105">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="1f3ec-106">有关详细信息，请参阅[委托](../programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="1f3ec-106">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="1f3ec-107">以下示例生成 CS0149：</span><span class="sxs-lookup"><span data-stu-id="1f3ec-107">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
