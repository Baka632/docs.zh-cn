# <a name="compiler-error-cs1575"></a><span data-ttu-id="b2a12-103">编译器错误 CS1575</span><span class="sxs-lookup"><span data-stu-id="b2a12-103">Compiler Error CS1575</span></span>
<span data-ttu-id="b2a12-104">Stackalloc 表达式要求在类型后有 []</span><span class="sxs-lookup"><span data-stu-id="b2a12-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="b2a12-105">使用 [stackalloc](../language-reference/operators/stackalloc.md)所请求的分配的大小必须在方括号中指定。</span><span class="sxs-lookup"><span data-stu-id="b2a12-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="b2a12-106">下面的示例生成 CS1575：</span><span class="sxs-lookup"><span data-stu-id="b2a12-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
