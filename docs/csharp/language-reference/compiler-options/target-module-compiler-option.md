-target:module

 <span data-ttu-id="ef1c3-112">Visual Studio 开发环境中不支持创建模块。</span><span class="sxs-lookup"><span data-stu-id="ef1c3-112">Creating a module is not supported in the Visual Studio development environment.</span></span>
