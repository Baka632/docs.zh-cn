<span data-ttu-id="f705d-256">某些方法（通常是异常帮助程序或其他实用工具方法）始终通过引发异常来退出。</span><span class="sxs-lookup"><span data-stu-id="f705d-256">Some methods, typically exception helpers or other utility methods, always exit by throwing an exception.</span></span> <span data-ttu-id="f705d-257">或者，帮助程序可以基于布尔参数的值引发异常。</span><span class="sxs-lookup"><span data-stu-id="f705d-257">Or, a helper may throw an exception based on the value of a Boolean argument.</span></span>
