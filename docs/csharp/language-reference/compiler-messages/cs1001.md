# <a name="compiler-error-cs1001"></a><span data-ttu-id="08ac6-103">编译器错误 CS1001</span><span class="sxs-lookup"><span data-stu-id="08ac6-103">Compiler Error CS1001</span></span>

<span data-ttu-id="08ac6-104">应输入标识符</span><span class="sxs-lookup"><span data-stu-id="08ac6-104">Identifier expected</span></span>

<span data-ttu-id="08ac6-105">没有提供标识符。</span><span class="sxs-lookup"><span data-stu-id="08ac6-105">You did not supply an identifier.</span></span> <span data-ttu-id="08ac6-106">标识符是所提供的类、结构、命名空间、方法、变量等的名称。</span><span class="sxs-lookup"><span data-stu-id="08ac6-106">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>

<span data-ttu-id="08ac6-107">以下示例声明一个简单的类，但没有为类指定名称：</span><span class="sxs-lookup"><span data-stu-id="08ac6-107">The following example declares a simple class but does not give the class a name:</span></span>

```csharp
public class //CS1001
{
    public int Num { get; set; }
    void MethodA() {}
}
```

<span data-ttu-id="08ac6-108">下面示例生成 CS1001，因为声明枚举时，必须指定成员：</span><span class="sxs-lookup"><span data-stu-id="08ac6-108">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>

```csharp
public class Program
{
    enum Colors
    {
        'a', 'b' // CS1001, 'a' is not a valid int identifier
        // The following line shows examples of valid identifiers:
        // Blue, Red, Orange
    };

    public static void Main()
    {
    }
}
```

<span data-ttu-id="08ac6-109">即使编译器不使用参数名（例如，在接口定义中），也需要参数名。</span><span class="sxs-lookup"><span data-stu-id="08ac6-109">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="08ac6-110">需要这些参数，以便使用接口的程序员对参数的含义有所了解。</span><span class="sxs-lookup"><span data-stu-id="08ac6-110">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>

```csharp
interface IMyTest
{
    void TestFunc1(int, int);  // CS1001
    // Use the following line instead:
    // void TestFunc1(int a, int b);
}

class CMyTest : IMyTest
{
    void IMyTest.TestFunc1(int a, int b)
    {
    }
}
```
