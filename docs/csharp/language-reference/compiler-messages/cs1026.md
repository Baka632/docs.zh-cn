 <span data-ttu-id="323be-105">找到不完整的语句。</span><span class="sxs-lookup"><span data-stu-id="323be-105">An incomplete statement was found.</span></span>  
  
 <span data-ttu-id="323be-106">此错误的一个常见原因是在 ASP.NET 页的内联表达式内放置的是一个语句，而不是一个表达式。</span><span class="sxs-lookup"><span data-stu-id="323be-106">A common cause of this error is placing a statement, rather than an expression, within an inline expression in an ASP.NET page.</span></span> <span data-ttu-id="323be-107">例如，以下示例错误：</span><span class="sxs-lookup"><span data-stu-id="323be-107">For example, the following is incorrect:</span></span>

  
 <span data-ttu-id="323be-108">以下示例正确：</span><span class="sxs-lookup"><span data-stu-id="323be-108">The following is correct:</span></span>

  
 <span data-ttu-id="323be-109">解释如下：</span><span class="sxs-lookup"><span data-stu-id="323be-109">It is interpreted as follows:</span></span>


```csharp  
   public static void Main()
```
