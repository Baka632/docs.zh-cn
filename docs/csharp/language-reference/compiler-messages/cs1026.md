# <a name="compiler-error-cs1026"></a><span data-ttu-id="5eca3-103">编译器错误 CS1026</span><span class="sxs-lookup"><span data-stu-id="5eca3-103">Compiler Error CS1026</span></span>

<span data-ttu-id="5eca3-104">) 应为</span><span class="sxs-lookup"><span data-stu-id="5eca3-104">) expected</span></span>  
  
 <span data-ttu-id="5eca3-105">找到不完整的语句。</span><span class="sxs-lookup"><span data-stu-id="5eca3-105">An incomplete statement was found.</span></span>  
  
 <span data-ttu-id="5eca3-106">此错误的一个常见原因是在 ASP.NET 页的内联表达式内放置的是一个语句，而不是一个表达式。</span><span class="sxs-lookup"><span data-stu-id="5eca3-106">A common cause of this error is placing a statement, rather than an expression, within an inline expression in an ASP.NET page.</span></span> <span data-ttu-id="5eca3-107">例如，以下示例错误：</span><span class="sxs-lookup"><span data-stu-id="5eca3-107">For example, the following is incorrect:</span></span>  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days;%>  
```  
  
 <span data-ttu-id="5eca3-108">以下示例正确：</span><span class="sxs-lookup"><span data-stu-id="5eca3-108">The following is correct:</span></span>  
  
```aspx-csharp  
<%=new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days %>  
```  
  
 <span data-ttu-id="5eca3-109">解释如下：</span><span class="sxs-lookup"><span data-stu-id="5eca3-109">It is interpreted as follows:</span></span>  
  
```aspx-csharp  
<% Response.Write(new TimeSpan(DateTime.Now.Ticks - new DateTime(2001, 1, 1).Ticks).Days); %>  
```  
  
 <span data-ttu-id="5eca3-110">下面的示例生成 CS1026：</span><span class="sxs-lookup"><span data-stu-id="5eca3-110">The following example generates CS1026:</span></span>  
  
```csharp

#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
