## <a name="example"></a><span data-ttu-id="f1578-111">示例</span><span class="sxs-lookup"><span data-stu-id="f1578-111">Example</span></span>

```csharp

class Test  
{  
    static int Main()

        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;
```
