# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="c4895-103">编译器警告（等级 2）CS0618</span><span class="sxs-lookup"><span data-stu-id="c4895-103">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="c4895-104">“member”已过时：“text”</span><span class="sxs-lookup"><span data-stu-id="c4895-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="c4895-105">类成员标记有 `Obsolete` 属性，这样，当引用该类成员时，将发出警告。</span><span class="sxs-lookup"><span data-stu-id="c4895-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="c4895-106">有关详细信息，请参阅[常用属性](../attributes/global.md)。</span><span class="sxs-lookup"><span data-stu-id="c4895-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="c4895-107">下面的示例生成 CS0618：</span><span class="sxs-lookup"><span data-stu-id="c4895-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
