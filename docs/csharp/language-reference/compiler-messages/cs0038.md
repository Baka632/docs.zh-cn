<span data-ttu-id="068f7-104">无法通过嵌套类型“type2”访问外部类型“type1”的非静态成员</span><span class="sxs-lookup"><span data-stu-id="068f7-104">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>

 <span data-ttu-id="068f7-105">类中的字段并不自动可用于嵌套类。</span><span class="sxs-lookup"><span data-stu-id="068f7-105">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="068f7-106">若要可用于嵌套类，该字段必须是[静态](../keywords/static.md)字段。</span><span class="sxs-lookup"><span data-stu-id="068f7-106">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="068f7-107">否则，必须创建外部类的实例。</span><span class="sxs-lookup"><span data-stu-id="068f7-107">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="068f7-108">有关详细信息，请参阅[嵌套类型](../../programming-guide/classes-and-structs/nested-types.md)。</span><span class="sxs-lookup"><span data-stu-id="068f7-108">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>

```csharp
// CS0038.cs

    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
