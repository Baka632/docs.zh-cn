 <span data-ttu-id="e0a8f-105">如果没有[不安全](../keywords/unsafe.md)的上下文，[sizeof](../operators/sizeof.md) 运算符只能用于大小为编译时常数的类型。</span><span class="sxs-lookup"><span data-stu-id="e0a8f-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="e0a8f-106">如果发生此错误，请使用不安全的上下文。</span><span class="sxs-lookup"><span data-stu-id="e0a8f-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="e0a8f-107">下面的示例生成 CS0233：</span><span class="sxs-lookup"><span data-stu-id="e0a8f-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
