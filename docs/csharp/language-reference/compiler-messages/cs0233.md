# <a name="compiler-error-cs0233"></a><span data-ttu-id="53397-103">编译器错误 CS0233</span><span class="sxs-lookup"><span data-stu-id="53397-103">Compiler Error CS0233</span></span>

<span data-ttu-id="53397-104">“identifier”没有预定义的大小，因此 sizeof 只能在不安全的上下文中使用</span><span class="sxs-lookup"><span data-stu-id="53397-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="53397-105">如果没有[不安全](../keywords/unsafe.md)的上下文，[sizeof](../operators/sizeof.md) 运算符只能用于大小为编译时常数的类型。</span><span class="sxs-lookup"><span data-stu-id="53397-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="53397-106">如果发生此错误，请使用不安全的上下文。</span><span class="sxs-lookup"><span data-stu-id="53397-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="53397-107">下面的示例生成 CS0233：</span><span class="sxs-lookup"><span data-stu-id="53397-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
