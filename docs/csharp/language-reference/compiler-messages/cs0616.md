# <a name="compiler-error-cs0616"></a><span data-ttu-id="8620c-103">编译器错误 CS0616</span><span class="sxs-lookup"><span data-stu-id="8620c-103">Compiler Error CS0616</span></span>

<span data-ttu-id="8620c-104">“class”不是特性类</span><span class="sxs-lookup"><span data-stu-id="8620c-104">'class' is not an attribute class</span></span>

## <a name="example"></a><span data-ttu-id="8620c-107">示例</span><span class="sxs-lookup"><span data-stu-id="8620c-107">Example</span></span>  

 <span data-ttu-id="8620c-108">下面的示例生成 CS0616。</span><span class="sxs-lookup"><span data-stu-id="8620c-108">The following sample generates CS0616.</span></span>

```csharp  
// CS0616.cs  
// compile with: /target:library

```  
  
## <a name="example"></a><span data-ttu-id="8620c-109">示例</span><span class="sxs-lookup"><span data-stu-id="8620c-109">Example</span></span>  

 <span data-ttu-id="8620c-110">下面的示例演示可以如何定义属性：</span><span class="sxs-lookup"><span data-stu-id="8620c-110">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
