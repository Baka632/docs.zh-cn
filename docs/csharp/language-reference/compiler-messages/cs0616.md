# <a name="compiler-error-cs0616"></a><span data-ttu-id="cbf61-103">编译器错误 CS0616</span><span class="sxs-lookup"><span data-stu-id="cbf61-103">Compiler Error CS0616</span></span>
<span data-ttu-id="cbf61-104">“class”不是特性类</span><span class="sxs-lookup"><span data-stu-id="cbf61-104">'class' is not an attribute class</span></span>  
  
 <span data-ttu-id="cbf61-105">尝试了在属性块中使用非特性类。</span><span class="sxs-lookup"><span data-stu-id="cbf61-105">An attempt was made to use a non-attribute class in an attribute block.</span></span> <span data-ttu-id="cbf61-106">所有属性类型都必须从 <xref:System.Attribute?displayProperty=nameWithType> 继承。</span><span class="sxs-lookup"><span data-stu-id="cbf61-106">All the attribute types need to be inherited from <xref:System.Attribute?displayProperty=nameWithType>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cbf61-107">示例</span><span class="sxs-lookup"><span data-stu-id="cbf61-107">Example</span></span>  
 <span data-ttu-id="cbf61-108">下面的示例生成 CS0616。</span><span class="sxs-lookup"><span data-stu-id="cbf61-108">The following sample generates CS0616.</span></span>  
  
```csharp  
// CS0616.cs  
// compile with: /target:library  
[CMyClass(i = 5)]   // CS0616  
public class CMyClass {}  
```  
  
## <a name="example"></a><span data-ttu-id="cbf61-109">示例</span><span class="sxs-lookup"><span data-stu-id="cbf61-109">Example</span></span>  
 <span data-ttu-id="cbf61-110">下面的示例演示可以如何定义属性：</span><span class="sxs-lookup"><span data-stu-id="cbf61-110">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
