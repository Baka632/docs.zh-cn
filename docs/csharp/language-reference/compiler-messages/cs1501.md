# <a name="compiler-error-cs1501"></a><span data-ttu-id="c1008-103">编译器错误 CS1501</span><span class="sxs-lookup"><span data-stu-id="c1008-103">Compiler Error CS1501</span></span>
<span data-ttu-id="c1008-104">方法“method”的重载不带有“number”个参数</span><span class="sxs-lookup"><span data-stu-id="c1008-104">No overload for method 'method' takes 'number' arguments</span></span>  
  
 <span data-ttu-id="c1008-105">对类方法进行了调用，但方法定义没有采用指定数目的自变量。</span><span class="sxs-lookup"><span data-stu-id="c1008-105">A call was made to a class method, but no definition of the method takes the specified number of arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c1008-106">示例</span><span class="sxs-lookup"><span data-stu-id="c1008-106">Example</span></span>  
 <span data-ttu-id="c1008-107">下面的示例生成 CS1501。</span><span class="sxs-lookup"><span data-stu-id="c1008-107">The following sample generates CS1501.</span></span>  
  
```csharp  
using System;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            ExampleClass ec = new ExampleClass();  
            ec.ExampleMethod();  
            ec.ExampleMethod(10);  
            // The following line causes compiler error CS1501 because
            // ExampleClass does not contain an ExampleMethod that takes  
            // two arguments.  
            ec.ExampleMethod(10, 20);  
        }  
    }  
  
    // ExampleClass contains two overloads for ExampleMethod. One of them
    // has no parameters and one has a single parameter.  
    class ExampleClass  
    {  
        public void ExampleMethod()  
        {  
            Console.WriteLine("Zero parameters");  
        }  
  
        public void ExampleMethod(int i)  
        {  
            Console.WriteLine("One integer parameter.");  
        }  
  
        //// To fix the error, you must add a method that takes two arguments.  
        //public void ExampleMethod (int i, int j)  
        //{  
        //    Console.WriteLine("Two integer parameters.");  
        //}  
    }  
}  
```
