## <a name="example"></a><span data-ttu-id="b986a-109">示例</span><span class="sxs-lookup"><span data-stu-id="b986a-109">Example</span></span>

 <span data-ttu-id="b986a-110">下面的示例生成 CS0592：</span><span class="sxs-lookup"><span data-stu-id="b986a-110">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
