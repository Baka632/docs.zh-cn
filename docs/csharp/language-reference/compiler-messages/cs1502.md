# <a name="compiler-error-cs1502"></a><span data-ttu-id="be6af-103">编译器错误 CS1502</span><span class="sxs-lookup"><span data-stu-id="be6af-103">Compiler Error CS1502</span></span>
<span data-ttu-id="be6af-104">与“name”最匹配的重载方法具有一些无效参数</span><span class="sxs-lookup"><span data-stu-id="be6af-104">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="be6af-105">正传递给方法的参数类型与该方法的参数类型不匹配时，会出现此错误。</span><span class="sxs-lookup"><span data-stu-id="be6af-105">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="be6af-106">如果重载调用的方法，则这些重载版本都不会具有正传递的参数类型相匹配的签名。</span><span class="sxs-lookup"><span data-stu-id="be6af-106">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="be6af-107">若要解决此问题，请执行下列操作之一：</span><span class="sxs-lookup"><span data-stu-id="be6af-107">To resolve this problem, do one of the following:</span></span>  
  
- <span data-ttu-id="be6af-108">仔细检查正在进行传递的参数的类型。</span><span class="sxs-lookup"><span data-stu-id="be6af-108">Double-check the types of the arguments being passed.</span></span> <span data-ttu-id="be6af-109">确保它们与正在调用的方法的参数匹配。</span><span class="sxs-lookup"><span data-stu-id="be6af-109">Make sure that they match the arguments of the method being called.</span></span>

- <span data-ttu-id="be6af-111">在适当的情况下，强制转换任何不匹配的参数，使它们与该方法需要的类型相匹配。</span><span class="sxs-lookup"><span data-stu-id="be6af-111">If appropriate, cast any mismatched parameters to match the type that the method is expecting.</span></span>
