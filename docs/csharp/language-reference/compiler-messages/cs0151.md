## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="98b77-107">不明确转换的示例</span><span class="sxs-lookup"><span data-stu-id="98b77-107">Example of ambiguous conversion</span></span>

<span data-ttu-id="98b77-108">在没有转换或者可用的隐式转换导致了不明确的情况时，会出现此错误。</span><span class="sxs-lookup"><span data-stu-id="98b77-108">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="98b77-109">下面的示例生成 CS0151：</span><span class="sxs-lookup"><span data-stu-id="98b77-109">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="98b77-110">Void 方法的示例</span><span class="sxs-lookup"><span data-stu-id="98b77-110">Example of void method</span></span>

<span data-ttu-id="98b77-111">[switch](../keywords/switch.md) 匹配表达式中的 [void](../builtin-types/void.md) 方法调用生成 CS0151。</span><span class="sxs-lookup"><span data-stu-id="98b77-111">A [void](../builtin-types/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="98b77-112">通过调用返回整型类型（如 [int](../builtin-types/integral-numeric-types.md) 或 [long](../builtin-types/integral-numeric-types.md)）的方法可修复该错误。</span><span class="sxs-lookup"><span data-stu-id="98b77-112">You can fix the error by calling a method that returns an integral type such as [int](../builtin-types/integral-numeric-types.md) or [long](../builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
