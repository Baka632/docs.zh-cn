# <a name="language-integrated-query-linq"></a><span data-ttu-id="f5e12-103">语言集成查询 (LINQ)</span><span class="sxs-lookup"><span data-stu-id="f5e12-103">Language Integrated Query (LINQ)</span></span>

<span data-ttu-id="f5e12-104">语言集成查询 (LINQ) 是一系列直接将查询功能集成到 C# 语言的技术统称。</span><span class="sxs-lookup"><span data-stu-id="f5e12-104">Language-Integrated Query (LINQ) is the name for a set of technologies based on the integration of query capabilities directly into the C# language.</span></span> <span data-ttu-id="f5e12-105">数据查询历来都表示为简单的字符串，没有编译时类型检查或 IntelliSense 支持。</span><span class="sxs-lookup"><span data-stu-id="f5e12-105">Traditionally, queries against data are expressed as simple strings without type checking at compile time or IntelliSense support.</span></span> <span data-ttu-id="f5e12-106">此外，需要针对每种类型的数据源了解不同的查询语言：SQL 数据库、XML 文档、各种 Web 服务等。</span><span class="sxs-lookup"><span data-stu-id="f5e12-106">Furthermore, you have to learn a different query language for each type of data source: SQL databases, XML documents, various Web services, and so on.</span></span> <span data-ttu-id="f5e12-107">借助 LINQ，查询成为了最高级的语言构造，就像类、方法和事件一样。</span><span class="sxs-lookup"><span data-stu-id="f5e12-107">With LINQ, a query is a first-class language construct, just like classes, methods, events.</span></span>

## <a name="query-expression-overview"></a><span data-ttu-id="f5e12-114">查询表达式概述</span><span class="sxs-lookup"><span data-stu-id="f5e12-114">Query expression overview</span></span>

- <span data-ttu-id="f5e12-115">查询表达式可用于查询并转换所有启用了 LINQ 的数据源中的数据。</span><span class="sxs-lookup"><span data-stu-id="f5e12-115">Query expressions can be used to query and to transform data from any LINQ-enabled data source.</span></span> <span data-ttu-id="f5e12-116">例如，通过一个查询即可检索 SQL 数据库中的数据，并生成 XML 流作为输出。</span><span class="sxs-lookup"><span data-stu-id="f5e12-116">For example, a single query can retrieve data from a SQL database, and produce an XML stream as output.</span></span>

- <span data-ttu-id="f5e12-117">查询表达式易于掌握，因为使用了许多熟悉的 C# 语言构造。</span><span class="sxs-lookup"><span data-stu-id="f5e12-117">Query expressions are easy to master because they use many familiar C# language constructs.</span></span>

- <span data-ttu-id="f5e12-118">查询表达式中的变量全都是强类型，尽管在许多情况下，无需显式提供类型，因为编译器可以推断出。</span><span class="sxs-lookup"><span data-stu-id="f5e12-118">The variables in a query expression are all strongly typed, although in many cases you do not have to provide the type explicitly because the compiler can infer it.</span></span> <span data-ttu-id="f5e12-119">有关详细信息，请参阅 [LINQ 查询操作中的类型关系](../programming-guide/concepts/linq/type-relationships-in-linq-query-operations.md)。</span><span class="sxs-lookup"><span data-stu-id="f5e12-119">For more information, see [Type relationships in LINQ query operations](../programming-guide/concepts/linq/type-relationships-in-linq-query-operations.md).</span></span>

- <span data-ttu-id="f5e12-120">只有在循环访问查询变量后，才会执行查询（例如，在 `foreach` 语句中）。</span><span class="sxs-lookup"><span data-stu-id="f5e12-120">A query is not executed until you iterate over the query variable, for example, in a `foreach` statement.</span></span> <span data-ttu-id="f5e12-121">有关详细信息，请参阅 [LINQ 查询简介](../programming-guide/concepts/linq/introduction-to-linq-queries.md)。</span><span class="sxs-lookup"><span data-stu-id="f5e12-121">For more information, see [Introduction to LINQ queries](../programming-guide/concepts/linq/introduction-to-linq-queries.md).</span></span>

- <span data-ttu-id="f5e12-122">在编译时，查询表达式根据 C# 规范规则转换成标准查询运算符方法调用。</span><span class="sxs-lookup"><span data-stu-id="f5e12-122">At compile time, query expressions are converted to Standard Query Operator method calls according to the rules set forth in the C# specification.</span></span> <span data-ttu-id="f5e12-123">可使用查询语法表示的任何查询都可以使用方法语法进行表示。</span><span class="sxs-lookup"><span data-stu-id="f5e12-123">Any query that can be expressed by using query syntax can also be expressed by using method syntax.</span></span> <span data-ttu-id="f5e12-124">不过，在大多数情况下，查询语法的可读性更高，也更为简洁。</span><span class="sxs-lookup"><span data-stu-id="f5e12-124">However, in most cases query syntax is more readable and concise.</span></span> <span data-ttu-id="f5e12-125">有关详细信息，请参阅 [C# 语言规范](~/_csharplang/spec/expressions.md#query-expressions)和[标准查询运算符概述](../programming-guide/concepts/linq/standard-query-operators-overview.md)。</span><span class="sxs-lookup"><span data-stu-id="f5e12-125">For more information, see [C# language specification](~/_csharplang/spec/expressions.md#query-expressions) and [Standard query operators overview](../programming-guide/concepts/linq/standard-query-operators-overview.md).</span></span>

- <span data-ttu-id="f5e12-126">通常，我们建议在编写 LINQ 查询时尽量使用查询语法，并在必要时尽可能使用方法语法。</span><span class="sxs-lookup"><span data-stu-id="f5e12-126">As a rule when you write LINQ queries, we recommend that you use query syntax whenever possible and method syntax whenever necessary.</span></span> <span data-ttu-id="f5e12-127">这两种不同的形式在语义或性能上毫无差异。</span><span class="sxs-lookup"><span data-stu-id="f5e12-127">There is no semantic or performance difference between the two different forms.</span></span> <span data-ttu-id="f5e12-128">查询表达式通常比使用方法语法编写的等同表达式更具可读性。</span><span class="sxs-lookup"><span data-stu-id="f5e12-128">Query expressions are often more readable than equivalent expressions written in method syntax.</span></span>

- <span data-ttu-id="f5e12-129">一些查询操作（如 <xref:System.Linq.Enumerable.Count%2A> 或 <xref:System.Linq.Enumerable.Max%2A>）没有等效的查询表达式子句，因此必须表示为方法调用。</span><span class="sxs-lookup"><span data-stu-id="f5e12-129">Some query operations, such as <xref:System.Linq.Enumerable.Count%2A> or <xref:System.Linq.Enumerable.Max%2A>, have no equivalent query expression clause and must therefore be expressed as a method call.</span></span> <span data-ttu-id="f5e12-130">可以各种方式结合使用方法语法和查询语法。</span><span class="sxs-lookup"><span data-stu-id="f5e12-130">Method syntax can be combined with query syntax in various ways.</span></span> <span data-ttu-id="f5e12-131">有关详细信息，请参阅 [LINQ 中的查询语法和方法语法](../programming-guide/concepts/linq/query-syntax-and-method-syntax-in-linq.md)。</span><span class="sxs-lookup"><span data-stu-id="f5e12-131">For more information, see [Query syntax and method syntax in LINQ](../programming-guide/concepts/linq/query-syntax-and-method-syntax-in-linq.md).</span></span>

- <span data-ttu-id="f5e12-132">查询表达式可被编译成表达式树或委托，具体视应用查询的类型而定。</span><span class="sxs-lookup"><span data-stu-id="f5e12-132">Query expressions can be compiled to expression trees or to delegates, depending on the type that the query is applied to.</span></span> <span data-ttu-id="f5e12-133"><xref:System.Collections.Generic.IEnumerable%601> 查询编译为委托。</span><span class="sxs-lookup"><span data-stu-id="f5e12-133"><xref:System.Collections.Generic.IEnumerable%601> queries are compiled to delegates.</span></span> <span data-ttu-id="f5e12-134"><xref:System.Linq.IQueryable> 和 <xref:System.Linq.IQueryable%601> 查询编译为表达式树。</span><span class="sxs-lookup"><span data-stu-id="f5e12-134"><xref:System.Linq.IQueryable> and <xref:System.Linq.IQueryable%601> queries are compiled to expression trees.</span></span> <span data-ttu-id="f5e12-135">有关详细信息，请参阅[表达式树](../expression-trees.md)。</span><span class="sxs-lookup"><span data-stu-id="f5e12-135">For more information, see [Expression trees](../expression-trees.md).</span></span>
